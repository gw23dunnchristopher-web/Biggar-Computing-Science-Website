Predefined Functions at Higher Level

At the National 5 level, we learned about three different predefined function and how to use them. These functions were:

FunctionDescriptionCode
RandomGenerates a random number between two values.random.randint(1,6)
RoundRounds a number to a specified number of decimal places.round(1.2345,2)
LengthReturns the length of a string or array.len("Hello")

At the Higher level, there are a few more predefined functions that we have to know about - ord(), chr(), int() and modulus.

Convert from character to ASCII and vice versa

The ord() and chr() functions can be used to convert a character to its equivalent ASCII value and vice versa as shown below:

FunctionReturned Value
ord("A")Returns 65
ord("!")Returns 33
chr(65)Returns "A"
chr(35)Returns "#"

Converting from Floating-Point Numbers to Integers

If the int() function is used on a floating-point number, it will remove everything after the decimal point.

For example:

FunctionReturned Value
int(1.2345)Returns 1
int(1.9999)Returns 1
int(-1.2345)Returns -1
int(-2.9999)Returns -2

Modulus

The modulus (or modulo) operation finds the remainder when one number is divided by another. For example, if you divide 10 by 3, the result is 3 with a remainder of 1.

Many programs have a separate pre-defined function that is used to perform a modulus operation. However, in Python it is represented by the '%' symbol, and performing a modulus operation is as easy as replacing the '/' symbol with '%' in a division.

For example:

CodeReturned Value
10 % 3Returns 1
10 % 2Returns 0
