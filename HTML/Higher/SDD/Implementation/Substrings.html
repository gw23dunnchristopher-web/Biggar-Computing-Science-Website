Substrings

What is a substring?

A substring is smaller part of a bigger string. If you have a long word or sentence, such as the word 'sunflower', you can extract a substring from it, for example the word 'sun' or 'flower', or even 'unflo'. Any of these would be considered substrings.

Many programming languages use pre-defined functions to extract substrings from strings. Python, however, treats strings as an array of characters, so it can use this fact to extract characters from the string in the same way you would use an index number to exract a value from an array.

Creating Substrings

Look at the example below:

name = "Mr. Dunn"

letter = name[0]

In this example, the variable letter will be assigned the value "M".

How this works

  • If we think of a string as an array, it places the first letter in element 0, the second letter in element 1 and so on.
01234567
Mr.Dunn
  • It can also count from the end of the string using negative numbers.
-8-7-6-5-4-3-2-1
Mr.Dunn
  • So if we entered the following code:

name = "Mr. Dunn"

letter = name[-1]

  • The variable letter would be assigned the value "n".
  • Additionally, we can use a colon to tell the program where we want to start and end. We type variableName[start:end-1]. Look at the examples below:
CodeValue placed in substring
substring = name[0:2]"Mr"
substring = name[4:7]"Dun"
substring = name[-3:0]"unn"

Note that the number to the right of the colon is the index of the last letter you want to display plus 1!

It should also be noted that if the character on the left side of the colon is left blank, it will default to 0. If the character on the right side is left blank, it will default to the last character in the string. For example:

substring = name[:3]

This would be the same as:

substring = name[0:3]

Both of these would assign the value "Mr" to the variable substring.

substring = name[2:]

Would be the same as:

substring = name[2:8]

Both of these would assign the value ". Dunn" to the variable substring.
