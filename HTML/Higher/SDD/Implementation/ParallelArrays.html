Parallel 1D Arrays

What are parallel arrays?

When we have two or more arrays that are related to each other, we call them parallel arrays. The arrays will be processed together, and are used to store related data in each index. For example, if we have three arrays - firstName, surname and prelimMarks, then the data in element 0 of the firstName array would hold a person's first name, the data in element 0 of the surname array would hold the same person's surname, and the data in element 0 of the prelimMarks array would hold the same person's prelim mark.

firstName = [""] * 19

surname = [""] * 19

prelimMarks = [0] * 19

This set of parallel arrays may contain information such as the data in the table below:

0 1 2 3 ... 19
firstName John Jane Bob Alice ... Elliot
surname Smith Doe Brown Johnson ... Anderson
prelimMarks 52 78 45 83 ... 76

Using Append

In Python, we can 'append' data to an array using the '.append()' function. This will add the data onto then end of the array, and it will increase the size of the array to do so if there is no aditional space. The function looks like this:

arrayName.append(data)

For example, if we wanted to add the name add Jim Butcher, who got a score of 64 in his prelim, to our parallel arrays, we could use the following code:

firstName.append("Jim")

surname.append("Butcher")

prelimMarks.append(64)

Looping through an Array with variable size

When we are using arrays that have a variable size, we can still use a for loop to loop through the array. You could used the len() function to get the length of the array and then loop through the array that many times. However, it is more efficient to do it this way instead:

for element in array:

# do something

For example if you have an array called firstName, you can loop through it like this:

for name in firstName:

print(name)
