Arrays of Records

What is a record in programming?

A record is a data structure that contains a group of values of different types. Each one of these values is referred to as an 'attribute'.

The record structure must be defined. For example, a record of a city might contain four attributes - cityName (string), population (integer), areaKm (integer) and capital (Boolean).

Python does not actually use records, instead we use Classes to simulate the use of records. For example:

from dataclasses import dataclass


@dataclass

class cityData:

cityName: str = ""

population: int = 0

areaKm: int = 0

capital: bool = False

Creating an Array of Records

An array of records is an array where each element is a record. For example, we could create an array of records called 'cities' that contains 10 records of the type 'cityData'.

We can create this using a combination of the code to initialise an array and a fixed loop. For example:

cities = [cityData() for x in range(10)]

Dot Notation

To access the attributes of a record, we use dot notation. This can be used to both read and write to the attributes. For example:

cities[0].cityName = "Edinburgh"

cities[0].population = 500000

cities[0].areaKm = 200

cities[0].capital = True


print(cities[0].cityName)

print(cities[0].population)

print(cities[0].areaKm)

print(cities[0].capital)

The above code will write the values 'Edinburgh', 500000, 200 and True, to the cityData record located in element 0 of the array, and then display the values held in the attributes of that element.

Note that when using dot notation, you use the name of the array, followed by the index number of the element you want to access in square brackets, followed by a dot the name of the attribute you want to access.

Writing to an Array of Records

A fixed loop can be used to write to an Array of Records. Just like with an ordinary array, you can either loop for the length of the array or, if it does not have a fixed length, the append function can be used.

For example, the following code will loop 10 times, and each time it will ask the user to enter the name, population, area and whether or not the city is a capital. It will then write these values to the attributes of the record in the current element of the array.

for x in range(10):

cities[x].cityName = input("Enter the name of the city: ")

cities[x].population = int(input("Enter the population of the city: "))

cities[x].areaKm = int(input("Enter the area of the city in km: "))

cities[x].capital = input("Is the city a capital? (True/False): ")

if cities[x].capital == "True":

cities[x].capital = True

else:

cities[x].capital = False
