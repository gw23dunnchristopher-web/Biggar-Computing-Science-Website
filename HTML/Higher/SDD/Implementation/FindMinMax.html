Find Minimum and Maximum

Find Minimum

This standard algorithm finds the smallest value in an array. If follows these steps:

  • Set the minimum value to the first value in the array.
  • Loop through the array, comparing each value to the current minimum.
  • If a value is smaller than the current minimum, set the minimum to that value.
  • After the loop has finished, the minimum value will be the smallest value in the array.

You can also use this method to find the position of the lowest number in the array by setting a position variable to 0 and then updating it with the current iteration number when lower values are found.

Pseudocode (SQA Reference Language)
Line 1SET min TO array[0]
Line 2SET position TO 0
Line 3FOR index FROM 1 TO end of array DO
Line 4IF array[index] < min THEN
Line 5SET min TO array[index]
Line 6SET position TO index
Line 7END IF
Line 8END FOR
Line 9SEND "The minimum value is " & min & " at position " & position TO DISPLAY
Python Code

min = array[0]

position = 0


for index in range(1, len(array)):

if array[index] < min:

min = array[index]

position = index


print("The minimum value is", min, "at position", position)

Find Maximum

This standard algorithm finds the largest value in an array. If follows these steps:

  • Set the maximum value to the first value in the array.
  • Loop through the array, comparing each value to the current maximum.
  • If a value is larger than the current maximum, set the maximum to that value.
  • After the loop has finished, the maximum value will be the largest value in the array.

Just as with the Find Minimum standard algorithm, the Find Maximum standard algorithm can be used to find the position of the largest value in the array.

Pseudocode (SQA Reference Language)
Line 1SET max TO array[0]
Line 2SET position TO 0
