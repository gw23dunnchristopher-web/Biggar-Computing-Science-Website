What is a Linear Search?

A linear search checks an array for the presence of a particular item of data. The user is asked to enter the data that they are searching for, and the program will then compare each item in the array to the data they entered. The program will continue whether or not the item is found - if it not found then a message will be displayed telling the user so.

Linear Search - Version 1

This algorithm searches the array until the target is found, or the entire array has been checked (used when the target will only appear once in the array (e.g. to find a particular pupil).

Pseudocode (SQA Reference Language)
Line 1 SET found TO FALSE
Line 2 SET index TO 0
Line 3 RECEIVE target FROM keyboard
Line 4 WHILE found = FALSE AND index < 10 DO
Line 5 IF array[index] = target THEN
Line 6 SET found TO TRUE
Line 7 SEND target & " found at position " & index TO DISPLAY
Line 8 ELSE
Line 9 SET index TO index + 1
Line 10 END IF
Line 11 END WHILE
Line 12 IF found = FALSE THEN
Line 13 SEND target & " not found." TO DISPLAY
Line 14 END IF
Python Code

found = False

index = 0

target = input("Enter the data you are searching for: ")


while found == False and index < 10:

if array[index] == target:

found = True

print(target, "found at position", index)

else:

index += 1


if found == False:

print(target, "not found.")

Linear Search - Version 2

This algorithm searches through the entire array and displays a message each time the target is found (used when the target may appear more than once in the array).

Pseudocode (SQA Reference Language)
Line 1 SET found TO FALSE
Line 2 RECEIVE target FROM KEYBOARD
Line 3 FOR index FROM 0 TO end of array DO
Line 4 IF array[index] = target THEN
Line 5 SET found TO TRUE
Line 6 SEND target & " found at position " & index TO DISPLAY
Line 7 END IF
Line 8 END FOR
Line 9 IF found = FALSE THEN
Line 10 SEND target & " not found." TO DISPLAY
Line 11 END IF
Python Code

found = False

target = input("Enter the data you are searching for: ")


for index in range(len(array)):

if array[index] == target:

found = True

print(target, "found at position", index)


if found == False:

print(target, "not found.")
