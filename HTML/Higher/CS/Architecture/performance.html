Factors Effecting Performance

Number of Processors (Cores)

  • There has been a development of several sets of processor components in one microprocessor.
  • A dual core processor has two separate CPUs in one chip and a quad core processor has four separate CPUs in one chip.
  • The more cores a processor has, the more sets of instructions the processor can receive and process at the same time.
  • For example, a quad-core processor can be used to improve load times for web apps containing client-side scripts or multimedia elements.
  • A dual core processor is not as fast as a single core processor running at twice the speed.

Width of the Data Bus

  • Data Bus - Set of parallel wires that connect the processor with memory and other hardware devices.
  • By increasing the data bus from 32 wires to 64 wires, the computer can transfer twice as much data at one time.
  • Increasing the size of the data bus improves the system performance of the computer
  • The amount of data that can be transferred in one cycle of the processor is known as the 'word length' of the bus.

Cache Memory

  • Cache memory is a very small amount of memory that is built into, or at least very close to, the processor.
  • It can be referred to as Level 1, Level 2 or Level 3, which indicates how close it is to the processor. Level 1 is built right into the processor itself.
  • Cache memory contains frequently used instructions and data.
  • The processor checks to see if an instruction is in cache memory first (it checks Level 1, then 2, then 3) and if it is not in cache memory it goes to retrieve it from main memory.
  • It is much faster to read and write from cache memory than it is from regular memory because it is so close to the processor cores, so using cache memory can lead to an increase in computer performance.
  • Cache hit - Data is found in the cache and can be quickly retrieved.
  • Cache miss - Data is not found in the cache and has to be retrieved from main memory, which is much slower.
  • A larger cache will result in more cache hits.

Clock Speed

  • The processor's clock generates a series of 'pulses' at a constant rate, allowing everything that the processor does to be kept in time with the clock.
  • One single pulse is known as a Clock Cycle.
  • The rate at which it generates these pulses is known as the Clock Speed.
  • The processor's clock speed is measured in MHz (Megahertz) or, more commonly, GHz (Gigahertz).
  • A 1 GHz processor will generate pulses one thousand million times a second.
  • Different instructions to the processer will take different amounts of clock cycles to fetch, but the higher your clock speed the faster the processor can do this.
  • Increasing the clock speed of the computer's processor will speed up the rate at which it can fetch instructions, improving performance.
  • You can overclock a processor, manually increasing the clock speed, but doing so will generate a lot of heat and may damage the processor.
