Graphic Representation

Bitmap Graphics vs Vector Graphics

Bitmapped Graphics

  • Stored as a grid of pixels where each pixel is represented by a binary number to indicate the colour of the pixel.
  • The file size is fixed, and only depends on the number of pixels used in the image.

Vector Graphics

  • Stores a list of objects (shapes) and thier attributes (position, line colour, line thickness, x and y coordinates).
  • The file size of the image can change based on the number of objects drawn in the image, the more objects there are the more instructions will be required and so the larger the file size will be.

Advantages of Bitmapped Graphics vs Vector Graphics

Each individual pixel in a bitmap graphic can be edited, this allows you to create some extremely detailed images.

Because you are just changing the colour of the pixels, adding more details or features to an image will not increase the file size. The file size is solely based on the number of pixels there are and the number of colours that each pixel can represent.

Due to the amount of detail they can have, bitmaps are used for images with high amounts of realism. Photographs are stored in bitmap format, for example.

Disadvantages of Bitmapped Graphics vs Vector Graphics

Bitmaps have a fixed resolution meaning that they can become blurry or pixellated when you enlarge them or zoom into them. On the other hand, Vector graphics are resolution independent - this means that the image will appear just as crisp and clean no matter how much you enlarge it. Vector images redraw objects every time you zoom in or change them in any way.

On a bitmap, you cannot move an individual piece of the image without leaving a white gap where it once was. Vector images, however, are layered - you can move individual objects without affecting any of the other other objects in the image. Objects can even be placed in front of or behind each other.

Operations such as resize, delete and move, which are available in Vector graphics, cannot be used as easily in bitmap graphics as only the pixels are edited.

File sizes for Bitmap graphics tend to be larger, as the size of the file is based on the number of pixels and the number of colours that each pixel can represent. Vector graphics have a smaller file size as the file size is based on the number of objects in the image.
