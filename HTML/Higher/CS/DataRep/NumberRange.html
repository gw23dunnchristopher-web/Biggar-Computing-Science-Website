Number Range

Positive Integers Number Range

Without using two's complement, we cannot represent negative numbers. If only considering positive numbers, then the minimum value we can have is 0 (when all the binary numbers are set to 0) and the maximum amount that we can have is when all the binary numbers are set to 1. Therefor the maxmimum amount would depend on the number of binary characters that we have - the number of bits used for the binary number.

For example, the maximum number you could get with 8 bits would be 255, which is 1111 1111 in binary (or 28 - 1). This means that the number range that you can get with an 8-bit number would be 0-255.

If you are only considering positive integers, you can calculate the maximum number of any number of bits using the following formula:

Minimum = 0

Maximum = 2n-1

Where n is the number of bits.

Negative to Positive - Two's Complement

If we introduce two's complement we can now represent a different range of numbers, with the minimum value being negative.

In two's complement, one of the bits is used as the sign bit, so this will deduct 1 from the number used to calculate the maximum number. For example, the maximum number you could get with 8 bits would be 127, which is 0111 1111 in binary (or 27-1).

To calculate the minumum number, we can take the maximum number and add 1, then make it negative. For example, the minimum number you could get with 8 bits would be -128, which is 1000 0000 in binary (or -27).

The range for 8 bits in two's complement would therefore be -128 to 127.

.

You can calculate the minimum and maximum using the following formula.

Minimum = -2(n-1)

Maximum = 2(n-1)-1

Worked Examples

Example 1

What is the number range for a 12-bit number?

Minimum = -2(12-1) = -211 = -2048

Maximum = 2(12-1)-1 = 211-1 = 2047

Number Range = -2048 to 2047

Example 2

What is the number range for a 16-bit number?

Minimum = -2(16-1) = -215 = -32768

Maximum = 2(16-1)-1 = 215-1 = 32767

Number Range = -32768 to 32767
