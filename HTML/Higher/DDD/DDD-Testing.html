SQL Testing Tables

At Higher level, you will be required to test the database to ensure that it works as expected. This will involve testing SQL operations, comparing the output that you would expect to the actual output produced by the database.

To do this you can produce a testing table, showing the type of test, the SQL statement used, the expected output and the actual output.

SELECT squad, MIN(time) AS [Slowest Time], MAX(time) AS [Fastest Time]

FROM Swimmer

 3

One mark for each error found:

Error 1MIN and MAX functions are the wrong way round OR ALIAS assigned to wrong function.
Error 2No wildcard operator on both sides of 1 (eg WHERE squad LIKE "%1%" or "*1*").
