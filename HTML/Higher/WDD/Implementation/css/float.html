Float

An element can be positioned on the left or right side of its container using the float property. This is useful for positioning images and other elements on the same line as text.

The float property has many values that it can be set to, but at the Higher level you should only be concerned with two of them:

  • float:left - the element is positioned on the left side of its container.
  • float:right - the element is positioned on the right side of its container.

Float Examples

The following examples will demonstrate how the float property is used to position an image.

Example 1

In the example below, the <img> element has been position on the right side of its container, the <header> element, using the float:right property.

HTML

<header>

<h1>School Activities</h1>

<img class="imageBanner" src="activitiesBanner.jpg" alt="Banner">

</header>
CSS

header h1 {font-family:Arial; font-size:60px; display:inline;}

.imageBanner {width:200px; height:80px; float:right;}

Note that the <header> element has been given the <display:inline> property so that it doesn't take up the full width of the container and the image can appear beside it.

Example 2

The float property can also be used to word-wrap text around an image, as demonstrated in the football club page below. The addition of the margins creates whitespace around the image, pushing the text away from it.

Look at the example below:

HTML

<section class="club">

<h1>School Football Club</h1>

<img src="footballClub.jpg" id="footballImg">

<p>Are you passionate about football? Do you dream of scoring the winning goal or making that game saving tackle? Whether you're a seasoned player or just starting out, the School Football Club is the perfect place for you!</p>

<p>Our club offers</p>

<ul>

<li><b>Weekly Training Sessions</b>: Improve your skills with professional coaching.</li>

<li><b>Friendly Matches</b>: Compete against other schools and local teams.</li>

<li><b>Team Spirit</b>: Build lasting friendships and learn the value of teamwork .</li>

<li><b>Fun Events</b>: Participate in football camps, tournaments and social gatherings .</li>

</ul>

<p>Come and be a part of our soccer family! Lace up your boots, bring your enthusiasm, and let's hit the field together!</p>

<p>For more information, visit our school website or contact Mr. Smith, our</p>

</section>
CSS

#footballImg {width:200px; float:right; margin:10px;}

Clear

The effect of floating elements can be removed using the clear property. This is useful if you want to stop text from wrapping around an image. It also helps to prevent elements from overlapping.

To ensure that the four main page elements - <header>, <nav>, <main> and <footer> - start a new line and remain unaffected by any float properties applied elsehwere on the page, the following CSS rule is used:

header, nav, main, footer {display:block; clear:both;}

If you do not do this then you will get instances of large elements overlapping onto other sections of your web page.
