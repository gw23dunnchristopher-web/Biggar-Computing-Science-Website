Grouping Selectors

Grouping selectors using commas reduces the amount of code required and is more efficient than writing each CSS declaration separately.

Seperate DeclarationsGrouped Declarations

header {margin-top:5px;}

footer {margin-top:5px;}

nav {margin-top:5px;}

main {margin-top:5px;}

h1 {font-family:arial;}

p {font-family:arial;}

header, footer , nav , main {margin-top:5px;}

h1 , p {font-family:arial;}

Descendant Selector

Descendant Selectors are used in CSS to tell the browser to modify elements that are specifically inside other elements. For example:

header h1 { font-size:80px; font-family:Impact; color:white; }

This CSS rule will modify the size, font and color of the <h1> tag, but only the <h1> tag that is inside the header tag. Note that you do not use a comma in between the different element names as you would with a grouping selector.

If there is a rule in CSS for all of the elements on the page with the same tag, using a descendant selector will override the rule for the tag inside the parent element. For example:

p { font-size: 20px; font-family:Arial; color:black; }

.blackbox p { color:white; }

In this example, all of the paragraph text on the page will be coloured black, but any paragraph text inside an element with the blackbox class will appear white. You can see this example below - note that even though the text is white rather than black, it retains all of the attributes from the original rule (it is still size 20 and in the Arial font).

Past Paper Question - 2022 Q15

Kelly runs a company offering driving lessons.

She wants a website where users can view her profile page and from there directly access individual pages about each of her instructors: Anne, Sarah, Jamal and Eilidh. The website should also detail the courses offered, a frequently asked questions page and a page where users can contact the company.

(d)

On the ‘Courses’ page there are a large number of different course packages that are offered. A sample of the HTML code used to implement this page is shown below.

<h1>We offer varying courses including:</h1>

<ul>

<li>Beginners</li>

<li>Refreshers</li>

</ul>


<h2>Course Details</h2>

<ol>

<li>Beginners Course</li>

<ul>

<li>Cost: £30 per lesson</li>

<li>Minimum Lessons: 2</li>

<li>Lesson Length: 1 hr</li>

</ul>

<li>Refreshers Course</li>

<ul>

<li>Cost: £25 per lesson</li>

<li>Minimum Lessons: 1</li>

<li>Lesson Length: 1.5 hr</li>

</ul>

</ol>

The CSS rules shown below are used to style the ‘Courses’ page.

ul {color: yellow;}

ol ul li{color: blue;}

Describe the effect of these CSS rules on this page.

 2

One mark for each of the bullet points:

  • All of the text inside the <ul> elements will be yellow.
  • All of the text inside the <ul> elements that are inside the <ol> elements will be blue.
