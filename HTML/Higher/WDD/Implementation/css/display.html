The Display Property

The display property is used in CSS to determine how an element is displayed on a webpage. It can be set to the following values:

  • display:block - the element takes up the entire width of the element that it is contained inside.
  • display:inline - the element only takes up as much width as it needs.
  • display:inline-block - the element takes up as much width as it needs, but can have padding and margins.
  • display:none - the element is not displayed.
  • display:grid - the element is split into columns.

The default value of the display property varies depending on the element. For example, the <p> and <h1> elements default to display:block, while the <a> element defaults to display:inline.

At higher level, you only need to be able to use the block, inline and none values.

Display:block and Display:inline

In the <header> element of the example website below, the <h1> heading uses display:inline. This allows the image to be floated level with the heading.

header h1 {font-family:arial; font-size:60px; display:inline;}

If display:block were used instead it would force the image to float on a new line.

Grid

You do not need to know this value as part of the Higher specifications, though it is useful for you to know if you wish to make your own websites.

The above example also uses display:grid to align the clickable buttons. The grid value allows you to display items as if you were displaying them on a table, making it easy to line them up or space them out. You can specify the number of columns in your grid element by using grid-template-columns in the CSS rule for the element where you want to use a grid.

Simply type the word auto after grid-template-columns: a number of times equal to the number of columns you want; this will give you evenly spaced columns. You can specify how wide columns should be instead using a percentage or a number of pixels.

.grid {

display:grid;

grid-template-columns: auto auto auto;

}
