Height and Width

Web page design and implementation involves creating areas that either have a fixed size of change size with the content or display (changing window size or resolution.

In the example website about Lawn Care, the <body> of each web page is set to a fixed width of 880px:

body {width: 880px;}

The height of the <header>, <nav> and <footer>] are all set to a fixed height, which will remain constant on every page of the website:

header {height: 100px;}

nav {height: 50px;}

footer {height: 50px;}

The reason for the odd numbers (880px rather than 900px) is because when you use padding, it increases the size of the lement to accomodate the padding. The body in this example has a padding of 10px. This creates 10 additional pixels on either side of the contents of the element, a total of 20 for the left and right sides:

20 + 880 = 900 pixels
