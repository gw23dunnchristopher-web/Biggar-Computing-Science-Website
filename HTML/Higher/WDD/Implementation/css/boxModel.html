The Box Model, Margins and Padding

What is the Box Model?

Margins and Padding are used to push content away from the outer and inner edges of an element.

The box model allows us to define the space between elements as shown below:

Margins and Padding

Margins▶

Using the margins property in CSS will determine the amount of space there is between elements. You can use margins in CSS with the following properties:

  • margin
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

These properties may have the values:

  • auto - the browser calculates the margin.
  • length - a fixed margin in px, cm, etc.
  • % - a percentage of the width of the containing element.

Simultaneous Declaration of Margins

You can declare all four margins at once using the margin property. The values are applied in the following order:

  • Top
  • Right
  • Bottom
  • Left

For example, the following code will set the top margin to 10px, the right margin to 20px, the bottom margin to 30px and the left margin to 40px.

margin: 10px 20px 30px 40px;

Just using one value with the margin property will set all of the margins to that value. For example:

margin: 10px;

You can also set the top and bottom margins to one value and the left and right margins to another value. For example:

margin: 10px 20px;

This will set the top and bottom margins to 10px and the left and right margins to 20px.

Margins Example

In the example website, CSS declarations for the main page elements (<header>, <nav>, <div> and <footer>) are used to separate out content when displayed.

The small gap at the top of each area is implemented by styling a margin of 5 pixels at the top of the four elements. For example:

nav, header, main, footer {margin-top: 5px;}

Auto Margins

Auto margins are used to position an element in the middle of the browser’s window or within another element. In the example website, the 900 pixel wide page <body> element is always positioned in the middle of the window:

Padding▶

Using the padding property increases the amount of space between the edge of the element and the content inside it. You can use padding in CSS with the following properties:

  • padding
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

These properties may have the following values:

  • length - a fixed padding in px, cm, etc.

Simultaneous Declaration of Padding

Just as with margin, you can declare padding on all four sides of the element at the same time using the code below:

padding: 10px 20px 30px 40px;

This will set the top padding to 10px, the right padding to 20px, the bottom padding to 30px and the left padding to 40px.

Similarly, using just one value will set all four sides to the same padding distance:

padding: 10px;

Example using Padding

We will use the following CSS code on the Lawn Care website:

header, div, main, footer, section {padding: 10px;}

You can see how the page would look with this code on the left, and how it would look without it on the right.

Universal Selector

In CSS, the universal selector (*) is used to select all elements on a page. It can be used to set the margin and padding to 0 for all elements as shown below.

* {margin:0; padding:0;}

This is useful as it removes the default margin and padding that browsers apply to elements.
