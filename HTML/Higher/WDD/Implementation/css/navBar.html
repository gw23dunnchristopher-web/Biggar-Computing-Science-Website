Horizontal Navigation Bar

The HTML 5 <nav> element is used to contain website navigational links. Essentially, it functions in the same way as a <div> tag, but it is a semantic tag that used for the navigation bar. To create a navigation bar you would first declare it as an unordered list of hyperlinks, as shown below:

<nav>

<ul>

<li><a href="index.html">Home</a></li>

<li><a href="about.html">Sport</a></li>

<li><a href="contact.html">Music</a></li>

<li><a href="contact.html">Study</a></li>

<li><a href="contact.html">Drama</a></li>

</ul>

</nav>

Styling the Navigation Bar

The look and feel of the navigation bar can then be created using CSS. Below is an example of the steps you would take to change the unordered list above into a horizontal navigation bar:

Step 1 - Remove the Bullet Points

changing the list-style-type property of the unordered list element to noneremoves the bullet points from the list:

nav ul {list-style-type:none;}

Step 2 - Position the List Items Horizontally

To ensure the list of hyperlinks is distributed horizontally, each list item should be floated to the left. This can be done by changing the float property of the list items to left:

nav ul li {float:left;}

Step 3 - Space Out List Items

To create space between each list item, a width is delcared and the link text is centered. This will move the text to the center of each block, creating buttons.

nav ul li {float:left; width:100px; text-align:center;}

Step 4 - Creating Clickable Boxes

At the moment, you would only be able to click on the words within the button to move to the new page, but you want to be able to click any part of the button! To do this you must set the <a> element to display as a block. This will make it take up the whole area of the list item, and ensure that the area around the words can also be clicked.

nav ul li a {display:block;}

Step 5 - Controlling the Vertical Alignment

The vertical position of the text within the button can be controlled by declaring the height of the <nav> element and including padding in the <a> element.

nav {height:50px;}

nav ul li a {display:block; padding-top:15px;}

Step 6 - Adding Interactive Colours

To make the navigation bar interactive, you can add different colours to the buttons when the user hovers over them and when they click on them. We can do this by using the :hover pseudo-class.

nav ul li a:hover {background-color:black; color:white;}

With this code, for example, the background of the button will turn black and the text will turn white when you hover the cursor over the button.

Step 7 - Removing the Underlines

You can remove the underlines from the hyperlinks by setting the text-decoration property to none.

nav ul li a {display:block; padding:8px; text-decoration:none;}
