JavaScript

JavaScript events are used to implement interactive web content. Events are placed withing HTML elements as shown below:

<button onclick="functionName() onmouseover="functionName()" onmouseout="functionName()">Click Me</button>

The inverted commas contain the action to be executed when the event takes place. Possible actions include:

  • hiding page elements.
  • revealing page elements.
  • changing the position of an element.
  • changing the colour of an element.
  • changing the size of an element.
  • changing the appearance of text.

Actions can be executed by:

  • referring to the element containing the JavaScript event.
  • referring to a different element, identified by an id.
  • calling a JavaScript function containing the actions.

Changing Graphic Sizes▶

Example 1

Inline JavaScript can be used to increase and decrease the size of an image as the mouse pointer passes over and off of it.

<img src="gas.png"

onmouseover="this.style.width='180px'; this.style.height='180px';"

onmouseover="this.style.width='150px'; this.style.height='150px';"

class = "mower">

This example contains two events - onmouseover and onmouseout. They are used to execute four actions:

  • width='180px'
  • height='180px'
  • width='150px'
  • height='150px'

Each action in a JavaScript statement is seperated by a semi-colon (;). Note that each action in the above element is preceded by this.style.

  • this - Meaning this element. In this example it refers to the image (<img>) element containing the event.
  • style - The style property is used to change the appearance of an element.

Example 2

JavaScript functions may be called to increase and decrease the size of a graphic, as a mouse pointer passes over and out of a graphic. Using a function means using less code in each individual element.

<img src="gas.png"

The JavaScript functions above are passed the parameter this, referring to the element containing the event displayLarger(this).

Creating Rollover Images▶

Example 1

Rollover images can be created using two JavaScript events. The first event (onmouseover) displays an alternate graphic when the mouse passes over the image element. The second event (onmouseout) displays the original graphic when the mouse pointer leaves the image.

<img src="gas.png"

onmouseover="this.src='PetrolMower.png';"

onmouseout="this.src='gas.png';">

The action src=' ' points the image element towards a given address.

Example 2

A rollover image can also be achieved using two functions. The first function is called using onmouseover. The second is called using onmouseout. Each function is passed the img element as a parameter using this. As before each function changes the source (src) of the image.

JavaScript

<script>

function changeImage(element, image) { element.src = image; }

</script>
HTML

<img src="gas.png"

onmouseover="changeImage(this, 'PetrolMower.png')"

onmouseout="changeImage(this, 'gas.png')">

Note that in this example, variables have been used in the JavaScript function so that you only have to use one function for all of the images. When the function is called, 'element' will be substituted for whatever image element is going to change (usually 'this') and 'image' is being substituted for the address of the image that it is going to switch to.

Dynamically Highlighting Text▶

To implement an action on a different page element, an id is required to identify the element the action will be performed on.

An element is identified using document.getElementById(' ').

The example below highlights the paragraph text when the mouse passes over the image next to the paragraph, the text will return to it's normal colour when the mouse moves away:

<img src="gas.png"

onmouseover= "document.getElementById('highlight1').style.color='yellow';"

onmouseout= "document.getElementById('highlight1').style.color='white';">

Hiding and Revealing Elements▶

When a web page loads, elements may be initially hidden by applying the CSS declaration display:none. This can be implemented using an inline or internal style, or by implementing an external style (using a class, as shown below):

.hidden {display:none;}

In the exanoke below, three sections containing text are hidden when the page loads using external CSS:

#Kentucky, #Fescue, #Bermuda {display:none;}

A JavaScript function is written in the html file, that will take in three IDs. It will show one of them by setting the display attribute to 'block', and hide the other two by setting their display attributes to 'none'.

Then each of the images will have the section containing the hidden text placed below it, and the onclick event will be used in the image to call the function. When the image is clicked, the webpage will show the text with an ID that matches the first one entered into the function.

JavaScript

<script>

function showText(showTextID, hidden1, hidden2) {

document.getElementById(showTextID).style.display = 'block';

document.getElementById(hidden1).style.display = 'none';

document.getElementById(hidden2).style.display = 'none';

}

</script>
HTML

<img src="Bluegrass.jpeg" onclick="showText('Kentucky', 'Fescue', 'Bermuda')">

<section id="Kentucky">

<p>Kentucky Bluegrass is known for its lush, dense, and

dark green turf. It has a fine texture and forms a

beautiful, soft lawn that is pleasant to walk on barefoot.

This grass thrives in cooler climates and full sun but is

not very drought-tolerant. It is often mixed with other

grasses like fescue to improve its shade tolerance.</p>

</section>
