Types of Website Structure

The design of a website should indicate the type of navigational structure that will be used — usually linear or hierarchical. The design should also show the direction of each of the links.

  • Linear Structure - This is also known as a sequential structure, because the webpages follow a sequence, one webpage links to the next a which links to the next and so on. These are traditionally used by businesses that don't have a lot of content. There are no subpages.
  • Hierarchical Structure - This is the most common type of structure. It has a home page which leads to a set of top-level pages. Each of these pages can then have subpages. This is the structure that we will be using at Higher Level.

List of Lawn Care Website Requirements▶

A new website is being created about lawn care. The client wants the website give users information on planting lawns, different types of grasses, lawnmowers and fertilizer.

The homepage will provide:

  • A short introduction to the website and lawn care.
  • Internal Hyperlinks to the following subpages - Planting Lawns, Grass Types, Lawnmowers and Fertilisers

The Planting Lawns page will contain:

  • Two videos - tutorials on sowing and turf laying.

The Lawnmowers page will contain:

  • Information on different types of lawnmower and how to select the right one.
  • A list of simple rules on how and when to cut lawn grass.
  • Photographs showing different types of lawnmower.

The Fertlisers page will contain:

  • Information on organic and inorganic fertilisers.
  • Links to pages that recommend organic and inorganic fertilisers.

The Grass Types page will contain:

  • Information on two different types of grass.
  • Information on how to select the right type of grass.

  • Horizontal lines show pages can all be accessed from each other.
  • Vertical lines - show pages that can only be accessed from one page and only link back to that page.

The pages on the navigational bar are referred to as Level 1 pages - they are coloured blue in the diagram below. The pages that can only be accessed from one of the pages on the navigational bar (the subpages) are known as Level 2 pages - show here in light purple. If the subpages had their own subpages, those would be Level 3 pages, and so on.

Past Paper Question - 2023 Q17

A new video sharing website is being developed. This site should allow users to upload new videos to the site, watch uploaded videos and comment on them. Users should sign into the site using a username and password.

Once signed in, users should be able to navigate to one of three pages — a profile page, a videos page and a settings page.

From the profile page, users should be able to navigate to a page where they can upload videos and read comments left by other users, a page where they can edit their profile and a page where they can edit the details of previously uploaded videos.

From the videos page, users should be able to view a page with the top ten trending videos and view a page showing the videos that they have watched most recently.
