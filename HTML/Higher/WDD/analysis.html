End-User and Functional Requirements

During the analysis stage of website development, the following requirements should be identified:

End-User Requirements

  • The end users are the people who are going to be using the website.
  • Their requirements are the tasks they expect to be able to do using the website.

Functional Requirements

  • These are the tasks that the website must be able to perform.
  • Also information that the system has to contain to be able to carry out its functions.

These requirements will help to:

  • clarify the design for each webpage.
  • identify the features to be implemented on the website (videos, audio, images, search bar, naviation bar, etc).
  • evaluate whether the website is fit for purpose after development is complete.

Example of End-User and Functional Requirements

Imaging that a new website is being created about lawn care. The client wants the website to give users information on planting lawns, different types of grasses, lawnmowers and fertiliser.

End-User Requirements

Users should be able to...

  • Navigate the website easily.
  • Find out information on two categories of grass with advice on which type to select.
  • Find out information about lawnmowers and how to select the right one for the right type of grass.
  • View photographs of different lawnmowers.
  • Find out information about organic and inorganic fertilisers.
  • Watch videos that will show them how to plant lawns - including sowing and turfing.

Functional Requirements

The website must contain:

  • a navigation bar that allows users to navigate the website easily.
  • a home page that provides internal links to different topic pages (Grass Types, Lawnmowers, fertiliser and Planting Lawns).
  • individual pages for each of the topics listed above - all of these should link back to the home page.
  • a heading on each page that clearly states what the page is about.
  • information on two different types of grass.
  • information on how to select the right type of grass.
  • images of different lawnmowers and information on each type.
  • video tutorials on sowing and turfing lawns.
  • information on organic and inorganic fertilisers.
  • links to pages that recommend organic and inorganic fertilisers.
  • Inorganic and Organic Fertiliser pages that link back to the main Fertiliser page.

Past Paper Question - 2022 Q17

A new video sharing website is being developed. This site should allow users to upload new videos to the site, watch uploaded videos and comment on them. Users should sign into the site using a username and password.

Once signed in, users should be able to navigate to one of three pages — a profile page, a videos page and a settings page.

From the profile page, users should be able to navigate to a page where they can upload videos and read comments left by other users, a page where they can edit their profile and a page where they can edit the details of previously uploaded videos.

From the videos page, users should be able to view a page with the top ten trending videos and view a page showing the videos that they have watched most recently.

(a)

State one functional requirement of this website.

 1

Award 1 mark for each bullet. Maximum 1 mark.

  • A form to:
    • Add/Read Comments
    • allow users to sign in.
    • allow users to upload videos or edit profiles.
  • An element to:
    • Upload videos
    • Watch videos
