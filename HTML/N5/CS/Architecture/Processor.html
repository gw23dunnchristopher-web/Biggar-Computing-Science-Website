Computer Architecture - Data Flow and the Processor

The Flow of Data in a Computer

Computers receive information from input devices, like the mouse or keyboard, this data is then sent to the processor for processing. The processor then sends the results back to output devices, such as a monitor or speakers.

While the data is being processed, the processor may have to retrieve data from memory or storage, and it also may have to send data to memory or storage.

The Processor

The processor is the central processing unit (CPU) of a computer, and is often called the 'brain' of the computer. It is responsible for executing instructions, performing calculations, and controlling the flow of data in the system. The processor fetches instructions from memory, decodes them, and then executes them, performing the necessary operations on data.

The processor actually consists of three main parts - the ALU, the Control Unit and the Registers.

ALU The Arithmetic Logic Unit (ALU) is responsible for performing arithmetic and logical operations on data. It is also responsible for performing comparisons.
Control Unit The Control Unit is responsible for controlling the flow of data within the computer. It is also responsible for managing the flow of data between different components of the computer.
Registers The Registers are small, temporary memory locations that are used to store data and instructions. They are also used to store the results of arithmetic and logical operations.
