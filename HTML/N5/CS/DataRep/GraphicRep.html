Data Representation - Graphics

Bitmap Graphics and Vector Graphics

There are two main types of computer images: Bitmap Graphics and Vector Graphics. Each has different uses and strengths.

Bitmap (Raster) Graphics

A bitmap image (also called a raster image) is made up of tiny squares called pixels. Each pixel has a specific colour, and together, they create the full image. The pixels are arranged in a grid which makes up the image, and to get the colour, each pixel is assigned a number of bits.

Bit depth The more bits that are assigned to each pixel, the greater the number colours can be represented - for example, if 8 bits are used 256 colours can be represented but if 16 bits are used for each pixel, 65,536 colours can be represented. An image that only contains black and white pixels only uses 1 bit per pixel. The bit can either be 1 (black) or 0 (white).
Large File Size As bitmap images have to store information on each pixel, the file size of a raw bitmap file is usually quite large.
Image Editing In a bit map file, the colour of each individual pixel can be changed.
Layers Only one layer of pixels are used, so you can't put parts of the graphic behind other parts of the graphic.
Resolution Dependant If you zoom into a bitmap image, then all it can do is enlarge the existing pixels. This means that zooming in to a bitmap image will give the image a blurry, blocky look, as shown in the picture below.
Detailed Bitmaps can be used for very detailed images, even photographs.

Vector Graphics

A vector image is made using mathematical formulas, not pixels. Instead of storing colours for each pixel, it stores information about shapes, lines, and curves.

Objects and Attributes A vector graphic file contains a list of objects and their attributes (characteristics). Some attributes that every object has are:
  • Co-ordinates - where the object is drawn.
  • Fill Colour - the colour inside the object.
  • Line Colour - the colour of the line used to draw the object.
  • Line Thickness - The width of the line used to draw the object.
Small File Size As the computer only has to stored instructions to draw the objects rather than information on every pixel, vector images usually have a smaller file size.
Resolution Independent When you zoom into a vector graphic, the computer simply redraws it. This means that it remains just as sharp no matter how much you zoom in.
Layers Objects remain separate when drawn, so you can move them around or even move them in front or behind each other.
Image Editing Images are edited by changing the object attributes.
Simple Images Vector graphics are usually used for simpler images, such as logos. This is because the images have to consist of individual objects.
