Queries - Search Criteria

Search Criteria

In the Query View, you can enter criteria for each field in the Criteria row. Click on a button below to see the criteria that you can use on a specific field type.

Text Fields

Criteria Description Example
Exact match Searches for a text field that contains this data. "Fiddich"
Does not match Shows all records that do not contain this data. Not "Fiddich"
Begins with Shows records where the specified field's data begins with a specific character. Like F*
Does not Begin with Shows records where the specified field's data does not begin with a specific character. Not Like F*
Contains specific text Shows records where the specified field's data contains specific text. Like "*Mark*"
Does not Contain specific text Shows records where the specified field's data does not contain specific text. Not Like "*Mark*"
Ends with Shows records where the specified field's data ends with a specific character. Like *h
Does not end with Shows records where the specified field's data does not end with a specific character. Not Like *h
In this list? Shows records where the specified field contains data in the list. In("Mark", "Colin", "Peter")

Number Fields

Criteria Description Example
Equal to Shows records where the specified field's data is equal to the number entered. 15
Not Equal to Shows records where the specified field's data is not equal to the number entered. Not 15
Greater than Shows records where the specified field's data is greater than the number entered. >15
Less than Shows records where the specified field's data is less than the number entered. <15
Greater than or equal to Shows records where the specified field's data is greater than or equal to the number entered. >=15
Less than or equal to Shows records where the specified field's data is less than or equal to the number entered. <=15
Between two values Shows records where the specified field's data is between two values. Between 13 and 17

Date/Time Fields

Dates and times require that you put a hash (#) at either side of the date you are looking for.

Criteria Description Example
Exact match Shows records where the specified field's data is equal to the date entered. #12/12/2023#
Do not match Shows records where the specified field's data is not equal to the date entered. Not #12/12/2023#
Before Shows records where the specified field's data is before the date entered. <#12/12/2023#
After Shows records where the specified field's data is after the date entered. >#12/12/2023#
Between two dates Shows records where the specified field's data is between two dates. Between #12/12/2023# And #12/12/2024#
In this list? Shows records where the specified field contains data in the list. In(#12/12/2023#, #12/12/2024#, #12/12/2025#)

Any Field Type

Criteria Description Example
Is Null Shows records where the specified field is empty. Is Null
Is Not Null Shows records where the specified field is not empty. Is Not Null
