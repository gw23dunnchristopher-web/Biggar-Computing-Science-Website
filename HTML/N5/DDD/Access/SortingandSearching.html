Sorting and Searching

What is sorting?

One of the useful features of a database is being able to organise the records based on the different fields. For example, in a database of students you might want to sort all of the records so that they appear alphabetically by last name.

What is searching?

In addition to sorting the records, you can also perform a search. This searches through the database and only shows records that meet certain criteria. For this reason, searching the database could also be called 'filtering' it, as it hides all of the records that don't containn the information that you're looking for. An example would be searching a database of students, you could have the database only show you the students who are in 4th year.

Simple vs Complex

When a search or sort is considered to be 'simple' it means that it only uses one field. For example, sorting all of the students in the database by their last name would be simple sort, but if you wanted to sort them by thier last name and then thier first name (just in case two students have the same last name), that would be complex sort.

When searching a database, if you use just one criteria to search through the records - for example, searching for all of the students with over 90% attendance, that would be a simple search. If you had two criteria though, for example searching for all of the students who are in 4th year and have an attendance of over 90%, that would be complex search.

Sorting and Searching Tutorials

Below are tutorials on how to sort and search a database in MS Access. For these tutorials, we will use a pre-constructed database of students. You can download this database by clicking on the button below.

Access Students Database

Click on either Sorting or Searching below to see the tutorial on that skill. Note that Searching is divided into Text and Number fields, this is because the options you get when searching different field types is slightly different.

  • Sorting

    In this tutorial, we will sort the records in the Students table by age.

    1. Download and open the Students Database. Once the database is open, open the Students table.

    2. Click on the Age field name to select that field.

    3. In the ribbon at the top of the screen, click on Ascending to sort the records alphabetically (A-Z) or in numerical order (1,2,3...), or click on Descending to sort it reverse alphabetically (Z-A) or in reverse numerical order (...3,2,1).

    4. To remove the sort, click on Remove Sort. This would allow you to perform another simple sort.

      However, if you wish to perform a complex sort, just select another field name and then click Ascending or Descending again.

  • Searching (Text Fields)

    In this tutorial, we will search the database for all students in S4.

    1. If you have not downloaded the Students Database, do so now. Open the database and open the datasheet view of the table you want to search, in this case the Students table.

    2. Click on the text field name that you want to filter. In this case, click on the Year field name.

    3. On the ribbon at the top of the screen, click on the Filter button.

    4. A menu will appear over the field you selected, in the menu click on 'text filters'.

    5. A submenu for text filters will appear with a list of search terms. For this tutorial, we will click on 'Equals'.

      Here are what each of these filters do:

      • Equals - Only shows records where the field contains the text you enter.
      • Does Not Equal - Only shows records where the field does not contain the text you enter.
      • Begins With - Only shows records where the data in the field begins with the text you enter.
      • Does Not Begin With - Only shows records where the data in the field does not begin with the text you enter.
      • Contains - Only shows records where the data in the field contains the text you enter.
      • Does Not Contain - Only shows records where the data in the field does not contain the text you enter.
      • Ends with - Only shows records where the data in the field ends with the text that you enter.
      • Does Not End With - Only shows records where the data in the field does not end with the text that you enter.

    6. In the box that appears, type in the text that you want to search for. In this case, we will type in 'S4'. Once you have typed in the text, click 'OK'.

    7. Now, the database will only show you the records where the Year field contains 'S4'.

      You can see the number of records in this search at the bottom of the window, in this case there are 33.

      Only one field has been filtered so for, so this is a Simple Search. You can filter another field in the same way to make this a complex search.

    8. If you want to perform another simple search, you will have to remove the filter. To do this, click on the 'Toggle Filter' button in the ribbon.

  • Searching (Number Fields)

    In this tutorial, we will search the database for all students who are 16 or older.

    1. If you have not downloaded the Students Database, do so now. Open the database and open the datasheet view of the table you want to search, in this case the Students table.

    2. Click on the text field name that you want to filter. In this case, click on the Age field name.

    3. On the ribbon at the top of the screen, click on the Filter button.

    4. A menu will appear over the field you selected, in the menu click on 'number filters'.

    5. A submenu for number filters will appear with a list of search terms. For this tutorial, we will click on 'Greater Than or Equal to'.

      Here are what each of these filters do:

      • Equals - Only shows records where the field contains the exact number you enter.
      • Does Not Equal - Only shows records where the field does not contain the exact number you enter.
      • Less Than or Equal to - Only shows records where number in the field is less than or equal to the number you enter.
      • Greater Than or Equal to - Only shows records where number in the field is greater than or equal to the number you enter.
      • Between - Only shows records where the number in the field is between the two numbers that you enter.

    6. In the box that appears, type in the number that you want to search for. In this case, we will type in '16', because we are searching for students who are 16 or over. Once you have typed in the number, click 'OK'.

    7. Now, the database will only show you the records where the Age field contains the number 16 or over.

      You can see the number of records in this search at the bottom of the window, in this case there are 54.

      Only one field has been filtered so for, so this is a Simple Search. You can filter another field in the same way to make this a complex search.

    8. If you want to perform another simple search, you will have to remove the filter. To do this, click on the 'Toggle Filter' button in the ribbon.
