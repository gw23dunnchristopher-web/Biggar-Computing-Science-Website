Creating a New Database in MS Access

What is MS Access?

MS (Microsoft) Access is a database management system (DBMS) that allows users to create, manage, and manipulate databases. It is part of the Microsoft Office suite and is widely used for creating and managing databases for small to medium-sized businesses and organizations.

In National 5 Database Design and Development, we will be using MS Access because it not only has a graphical user interface, making it a little easier to understand, but also access to a SQL view, allowing us to perform SQL queries in it as well.

In the MS Access Tutorials on this site, you will see how to create a new database, add and delete records, use data validation and create queries using the MS Access GUI. First, we will look at the creation of a new database with two tables.

Creating a New Database

  1. Open MS Access - if you click on the Start button on your desktop (it looks like the Windows symbol) and 'Access' into the search bar, you should see it appear. Click on it to open it. The symbol for access is displayed here.

  2. Blank Database - Once access is open, you will have to create a blank database. You can click on the icon to the right in order to do so. Any databases you were working on that you have saved will also be listed on this screen or, if not, you can click on open to navigate to them.

  3. Name your database - You will now have to give your database a name. In this example, as we are creating the database for Items and Designers, we will give the database the name 'Designs'.

  4. Save your database in the right folder - By click on the small folder next to the name, you will be able to choose where to save your database. You should save it in your own 'Computing' folder.

  5. Click the Create button - Once you have chosen the right folder to save your database in, you can click the Create button to create it.

  6. Go to the Design View - A blank datasheet will appear. MS Access automatically creates one blank table for you. Go to the Design view by clicking on the Design View Icon in the top left of the screen.

  7. Give the table a name - When entering the Design View of a new table for the first time, you will be asked to give the table a name. We will call this table 'Designers'. After you name the table, you can click OK to open the Design View.

  8. Add field names - This is the design view, it will list the fields your table will use and their properties. It will start with a primary key field called 'ID', you can see that this is a primary key field because of the small key next to it. ID will be automatically be given the type 'Autonumber'. We are going to change this to 'Designer ID' and set the field type to 'Short Text'. We will also add the following fields below it - Designer Name, Email and Phone Number. All of these will also be Short Text fields.

    Short Text fields have a maximum of 255 characters, it is better to use these than Long Text, that can store up to a gigabyte of data! This is because using Short Text will make your database take up less space.

    Additional, if you want to delete a field, just right-click on it and click 'Delete Rows'.

  9. Add field lengths - You should add a field length for any text fields. The length of the text field can be found in its properties - whenever you click on one of the field names in the design view, you will be able to see the properties of that field at the bottom of the screen (for more information on the properties of different field types, go to the Field Properties page). The field size is the length of the field, and it is set to the maximum of 255 by default, but you can delete this and change it. As per our data dictionary, set the length of the Designer ID to 6, Designer Name to 20, Email to 30 and Phone Number to 11.

  10. Return to the Datasheet View - You can now click on the Datasheet icon in the top left to return to the Datasheet view. You will be asked to save your table when you do this, and can click 'Yes' to do so. Once in the datasheet view, you can type in data to add records to your database.

  11. Create the Second Table - Our database has two tables, Designers and Items. We have created the Designers Table, now we should also create the Items table. Click on the Create tab at the top of the screen, then click on the Table Icon.

  12. Go to the Design View and Name the Second Table - Click on the Design View Icon again and it will ask you to name the new table. Call this one 'Items' and click OK to open the Design View.

  13. Add field names and lengths to the Second Table - We will now add the fields to the second table in the Design View. As per our Data Dictionary, these fields are:

    • Item ID (Number, this will replace the default ID field)
    • Item Name (Short Text, 20)
    • Description (Short Text, 50)
    • Price (Number, but Access has a Currency type which is better for money)
    • Type (Short Text, 10)
    • Amount (Number)
    • Designer ID (Short Text, 6)
    Remember to adjust the field size for each of the Short Text fields in the properties for each field.

  14. Finally, you can click on the Datasheet View Icon to return to the Datasheet View. You will be asked to save the table, click 'Yes' to do so. Now you should have both tables. At this point you could start entering records, but we have some Data Validation to add first. This will be covered in the next tutorial.
