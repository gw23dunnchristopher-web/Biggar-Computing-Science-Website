Inserting a single new record

A new Screwdriver has arrived from Craft Supplies (ManufacturerID 441), it will be given the productCode 'SD20', will cost £2.99 and is not on order at the moment, but there are 24 in stock. This should be added to the database.

Step Action Code
1 Specify the table that you want to add the record to, if you are going to add data to every field you do not need to specify the fields.

INSERT INTO Product (productName, productCode, numberInStock, onOrder, costPrice, manufacturerID)
2 Specify the values that you want to add to the database. If you do not specify the fields, then these values must be in the order the fields appear in your database.

INSERT INTO Product (productCode, numberInStock, onOrder, costPrice, manufacturerID)

VALUES ('Screwdriver', 'SD20', 24, False, 2.99, 441);
  • onOrder is a Boolean field, in SQL this is represented by True or False. True to put a tick in the checkbox, and False not to. You can also use 1 for True and 0 for False.
  • Remember that if you are not going to be entering data into all of the fields, you must specify which fields you will be entering data into on the first line and then enter them in order on the VALUES line.

Inserting multiple new records

This method works in MySQL or SandboxSQL, but it does not work in MS Access. Unfortunately in MS Access you can only add one record at a time with INSERT INTO.

Imagine that you had to add the following records to the Product table:

productName productCode numberInStock onOrder costPrice manufacturerID
Nail (Size 8d) N08D 1000 0 0.05 531
Nail (Size 10d) N10D 800 0 0.10 531
Nail (Size 12d) N12D 600 0 0.15 531

You could write three separate INSERT INTO statements, but there is a quicker way. You can write one INSERT INTO statement and then specify the values for each record in brackets. Like so:

Step Action Code
1 Specify the table that you want to add the record to, if you are going to add data to every field you do not need to specify the fields.

INSERT INTO Product (productCode, numberInStock, onOrder, costPrice, manufacturerID)
2 Specify the values that you want to add to the database. You can add the additional records in brackets below the first - note that each new record is separated by a comma (,).

INSERT INTO Product (productCode, numberInStock, onOrder, costPrice, manufacturerID)

VALUES ('Nail (Size 8d)', 'N08D', 1000, 0, 0.05, 531),

('Nail (Size 10d)', 'N10D', 800, 0, 0.10, 531),

('Nail (Size 12d)', 'N12D', 600, 0, 0.15, 531);
