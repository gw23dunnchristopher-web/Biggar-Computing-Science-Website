SELECT SQL Statements

SELECT statements are used to retrieve data from a database. A SELECT statement consists of 4 main parts:

  1. The fields that you want to display in your results.
  2. The table or tables that you want to retrieve the data from.
  3. The criteria for your search.
  4. The order that you want the results to be displayed in.

SELECT - One Condition

Show the first name, last name, and gender of each student in S4 and display them in alphabetical order by last name.

Step Action Code
1 Select the fields that you want to display.

SELECT first_name, last_name, gender
2 Select the table(s) that you want to retrieve the data from.

SELECT first_name, last_name, gender

FROM Students
3 Enter the criteria for your search.

SELECT first_name, last_name, gender

FROM Students

WHERE Year = 'S4'
4 Enter the order that you want the results to be displayed in.

SELECT first_name, last_name, gender

FROM Students

WHERE year = 'S4'

ORDER BY last_name ASC;
  • Note that 'S4' has been entered in inverted commas, this is because S4 is text. You can also use speech marks.
  • ORDER BY will tell the program to organise the data in order by one of the fields. It can be organised either in Ascending (ASC) or Descending (DESC) order.
  • The ORDER BY statement has a semi-colon at the end, this is because it is the last line in the query.

SELECT - More than one condition

We want to show the First Name, Last Name and Year of all of the students who are over 15 in house Fiddich, the students should be ordered by age from oldest to youngest.

Step Action Code
1 Select the fields that you want to display.

SELECT first_name, last_name, year
2 Select the table(s) that you want to retrieve the data from.

SELECT first_name, last_name, year

FROM Students
3 Enter the criteria for your search.

SELECT first_name, last_name, year

FROM Students

WHERE age > 15 AND house = 'Fiddich'
4 Enter the order that you want the results to be displayed in.

SELECT first_name, last_name, year

FROM Students

WHERE age > 15 AND house = 'Fiddich'

ORDER BY age DESC;
  • Here DESC has been used to display the students in reverse order of age, so from oldest to youngest.
  • AND has been used to add another condition to the WHERE statement. You can add as many conditions as you want to the WHERE statement, just make sure to use AND between each one.
  • OR can be used instead of AND, but instead of displaying records that meet all of the conditions, it will display records that meet any of the conditions.
  • You can also use OR and AND together in the same query, just make sure to use brackets to group the conditions you want together.
