Flowcharts

A flowchart is a graphical notation style that uses shapes and text to show the steps in the design for a program - this can be easier to understand than pseudocode. Flowchats use symbols that match programming constructs such as selection and repitition. At the N5 level you need to be able to read a flowchart.

Flowchart Symbols

Symbol Name Meaning
Flow line Shows the direction between symbols
Terminal Used at the “start” and “end” of a design
Initialisation Used to declare variables or assign an initial value
Input/output Shows input or output
Decision Program may take different actions depending on conditions
Process Used for any a process e.g. a calculation
Predefined process Used for a predefined function
On-page connector May be used to split a flowchart to keep it on a single page.
