User Interface Design

What is a User Interface?

A user interface (UI) is the part of a computer, app, or website that you interact with. It's everything you see on the screen and how you use it to make the program do what you want.

There are three different types of user interface:

User Interface Description Example
Graphical User Interface (GUI) A GUI uses pictures, icons, buttons, and other visual elements to allow you to interact with a computer or program. You click or tap on icons and buttons to perform tasks. It's very visual and user-friendly.
Menu Driven Interface (MDI) A menu-driven interface is a type of user interface where users interact with a program or system through a series of menus. These menus present options or commands that the user can select, typically through the use of a pointer, keyboard, or touchscreen
Command Line Interface (CLI) A CLI is a text-based interface where you type commands to interact with the computer or program. There are no pictures or buttons, just lines of text. You type specific commands (like "dir" to list files or "cd" to change directories) and hit enter to perform actions. It’s less visual, but it gives you full control if you know the commands.

Any questions you get regarding a user interface at the N5 level will likely be asking you to create a GUI for a program.

User Interface Design

User Interfaces are designed at this stage to show the inputs and outputs of a program and how it will look to the user. Typically they are designed using a wireframe, or a sketch of the user interface.
