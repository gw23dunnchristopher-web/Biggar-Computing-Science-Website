Analysis

During the analysis phase we need to identify the purpose of the program and its functional requirements.

The purpose of the program is simply what the program will do, but the functional requirements can be broken down into three parts: input, process and output.

Functional Requirements

  • Input - This is the data that the user will need to enter into the program so that it can perform its function.
  • Processes - For this you describe the tasks that the program must carry out, such as what it does with the data that was entered.
  • Output - This describes what information will be displayed by the program.

Example - Car Speed

You have been asked to create a program that will calculate the speed of a moving car using the formula 'speed = distance / time'. The program will take in the distance the car has travelled in meters and the time it took to travel that distance in seconds. It will then calculate the speed of the car in m/s and display the result. List the inputs, processes and outputs for this program.

Input

Distance in meters

Time in seconds
Processes

Calculate the speed using (speed = distance / time)
Output

Speed in m/s

Example 2 - Pupil Usernames

A program is being created for the school to create a list of usernames for pupils. The program should ask the user to enter the number of pupils, the first name and last name of each pupil and their age. The program needs to make sure that each age entered is between 14 and 18, will then create and display the usernames in the following format (first name + last name + age). List the inputs, processes and outputs for this progam.

Show Answer
Input

Number of pupils

first name of each pupil

last name of each pupil

age of each pupil
Processes

Check to make sure the age entered is between 14 and 18

display an error message if it is not

create a username for each pupil (first name + last name + age)

add the username to a list
Output

List of usernames

Example 3 - Book Store Program

You are creating a simple program for an online bookstore that allows a customer to choose books, calculate the total price, and display the order details. The program should let the customer select books, enter the number of books they want to buy, and then calculate the total cost, with each book priced at £10. If the customer buys more than 5 books, a 10% discount is applied. Finally, the program should display the final total cost to the customer. Your task is to identify the inputs, processes, and outputs of this program.

Show Answer
Input

Number of books the customer wants to buy.
Processes

Multiply the number of books by £10 to get the total cost.

If the number of books is more than 5, apply a 10% discount (multiply the total by 0.9)
Output

The total cost of the books after the discount is applied.
