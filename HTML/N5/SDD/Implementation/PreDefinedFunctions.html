Pre-defined Functions

What is a Pre-defined Function?

Functions are blocks of code that performs specific tasks. The block of code is given a name and can be called and used elsewhere in the program. For example, print() and input() are functions. When we use these, the progam looks at the library where the code for the function is located. We can make our own functions, and we will at Higher level, but at N5 we will make use of some of the pre- defined functions in Python.

Arguments are the values that are passed to the function. For example, when we use the print() function, we can pass it a string, or a number, or a variable, which will be displayed to the user. When we use the input() function, we can pass it a question, which will be displayed to the pass it a question, which will be displayed to the user.

Pre-defined functions are also known as built-in functions because they come with the programming language, you don't have to create them yourself. They tend to perform common tasks, such as displaying text in the case of print() and receiving input from the user in the case of input().

In N5, we will learn to use the following pre-defined functions:

  • Round - Rounds a number to a specified decimal place.
  • Random - Generates a random number.
  • Length - Counts the number of characters in a string or elements in an array and returns that as a number.

Round

The round() function rounds a number to a specified decimal place. It takes two arguments, the number to be rounded and the number of decimal places to round to, separated by a comma.

For example, if you typed round(3.14525062, 2) it would return 3.15. This value will have to be stored in a variable or displayed using the print() function.

The code below adds the marks for an English test, a Maths test and a Computing test and divides it by 3 to get the average. The average is then rounded to 2 decimal places.

print("Enter your English mark:")

english = int(input())

print("Enter your Maths mark:")

maths = int(input())

print("Enter your Computing mark:")

computing = int(input())


average = (english + maths + computing) / 3

print("Your average mark is " + str(round(average, 2)))

Random

There are several different functions that generate a random number in Python. To use any of them you will need to import the 'random' library, a module containing code for the random number generating functions. To import the entire random library, you just type import random at the beginning of the program.

Importing the entire library will mean specifying the library whenever you use any of its functions, like this: random.randint()

For N5, however, we just need to generate an integer, so we might as well import the randint function from the random library. We can do that by placing: from random import randint

This will allow us to just called the randint() function directly - randint()

The randint() function takes 2 arguments, the lowest number in the range of numbers it can choose from, and the highest number. For example, typing randint(1,10) would return a random integer between 1 and 10.

The code below generates two random numbers between 1 and 6 and then adds them together.

from random import randint


dice1 = randint(1,6)

dice2 = randint(1,6)


total = dice1 + dice2


print("You rolled a " + str(dice1) + " and a " + str(dice2) + " for a total of " + str(total))

You can try this program in the code window below, try changing the numbers passed into randint to change the range of the random number.

Length

The length function returns the number of characters in a string or the number of elements in an array. This is useful for checking the length of a password, or using a fixed loop to repeat a block of code a number of times equal to the number of elements in an array.

Python uses the len() for this. For example, if you typed len("Hello") it would return 5.

The code below asks the user to enter a password, and checks that it is at least 8 characters long.

print("Enter a password:")

password = input()


passLength = len(password)


while len(password) < 8:

  print("Your password must be at least 8 characters.")

  print("Please try again:")

  password = input()


print("Your password is " + str(passLength) + " characters long.")
