Loops

In programming, a loop is a control structure that allows a set of instructions to be executed repeatedly based on a condition or a specific number of times. Loops are used when you need to perform the same task multiple times without writing the same code over and over.

  • Fixed Loop - A loop that repeats a specific number of times.
  • Conditional Loop - A loop that repeats based on a condition (e.g. it repeats either while a condition is true or until a condition is met.

Fixed Loops

In Python a fixed loop is also known as a FOR loop. Python uses the range() function to determine the number of times that the loop repeats - range(startValue, endValue - 1)

Fixed (FOR) Loop Examples
for x in range(0,3): This code will repeat 3 times, starting at 0 and ending at 2 (0, 1, 2).
for x in range(1,3): This code will repeat 2 times, starting at 1 and ending at 2 (1, 2).
for x in range(4): If a starting value is not specified, then the program will assume that it is 0. This code will repeat 4 times (0, 1, 2, 3)
for x in range(days): This program will use whatever value is in the variable ‘days’ to determine the number of times the loop repeats.
  • The ‘x’ value in this code is known as the ‘iteration number’.
  • It does not need to be x, any variable name will do
  • When the loop executes for the first time, the iteration number will be set to the starting value in the range function.
  • Each time the loop runs, the iteration number will be increased by 1.
  • The iteration number can be used inside the loop as a variable (displayed using the print function) to show how many times the loop has executed. This can also be useful to number the questions being asked in the loop.

Conditional Loops

In Python a Conditional Loop is known as a WHILE loop. In a WHILE loop, the loop will continue to repeat the cod inside it while a specificed condition is true. For more information on the kind of conditions you can use, please refer to the Selection section of this website.

Conditional (WHILE) Loop Examples

while password != "button20":

  print("Password Incorrect!")

  print("Please enter password:")

  password = input()

 This while loop continues to ask the user for their password until the user types "Button20"

counter = 0

while counter < 10:

  print(counter)

  counter = counter + 1

 This while loop emulates a fixed loop. The counter will increase by 1 ever time the code in the loop executes. The program will only exit the loop when the counter is 11 or more.

while number < 0 or number > 100:

  print("Please enter a number between 0 and 100!")

  number = int(input())

 This loop will continue to ask the user to enter a number until they enter a number that is between 0 and 100.
  • These loops are used when it is not known how many times the loop will be executed each time that the program is run
  • The code in the loop is executed while the condition in the while loop is true
  • The condition is at the start of the loop
  • The loop may not be executed at all - if the condition is false to start with
  • The loop may be executed once or many times – while the condition is true
