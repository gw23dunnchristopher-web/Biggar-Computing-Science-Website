Running Total (within Loop)

What is a running total?

Keeping a running total is a common task in programming. It is when you add up a series of numbers and keep a running total of the result. For example, if you want to increase a count by 1 every time a button is pressed, you can use a running total to keep a running count of the number of times.

To keep a running total in programming, we use a loop.

Running Total Examples

To keep a running total, you first have to tell the program what number you want to start with. This usually involves setting the number contained in the variable storing your total to 0. After that, Python has two ways to add to this total.

  • total = total + (number)
  • total += (number)

In both cases here, number can be replaced with the value being added onto the total. The pseudocode for keeping a running total would look something like this:

1.Set total to 0
2.Start fixed loop:
3.Get number from user
4.Add number to total
5.End fixed loop

Example 1 - Keeping a total of numbers entered by the user

This example will keep a running total of the numbers entered by the user. The program will keep asking the user for a number, which it will then add to the total. It will loop 5 times. The program will then display the total at the end.

total = 0


for x in range(5):

print("Enter a number:")

number = int(input())

total = total + number


print("The total is " + str(total))

Example 2 - Keeping count in a while loop

The following example will add 1 to a counter over and over until you type "Yes" when asked if you want to stop. This time, the running total uses a conditional (while) loop instead of a fixed loop.

counter = 0

stop = False


while stop == False:

counter += 1

print("The current count is " + str(counter))

print("Do you want to stop?")

answer = input()

if answer == "Yes" or answer == "yes" or answer == "y":

stop = True


print("The counter is " + str(counter))
