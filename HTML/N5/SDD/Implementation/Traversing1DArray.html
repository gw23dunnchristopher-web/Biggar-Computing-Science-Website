Traversing a 1D Array

What does it mean?

Traversing a 1D array is a standard algorithm that uses a loop to either allow users to enter data into an array or check data that is in an array. This data can then be displayed or used for other purposes. The pseudocode would look like this:

1.Start fixed loop
2.Get data from user and store in array in current iteration number index.
3.End fixed loop

Traversing a 1D array Examples

The following example shows how to traverse a 1D array and use the contents of the array.

Example 1 - Enter marks into an array and display them

This example has the user enter 6 test marks into an array. The program will then display them in a list.

marks = [0] * 6


for x in range(6):

print("Enter mark for test " + str(x+1) + ":")

marks[x] = int(input())


for x in range(6):

print("Test " + str(x+1) + " = " + str(marks[x]))

Example 2 - Which Scores are 50 or more?

This example looks at an array containing scores, as it moves through the array it will count all the scores that are over 50.

scores = [18,56,32,65,77,14,23,78,81]

counter = 0


for x in range(len(scores)):

if scores[x] > 50:

counter = counter + 1


print("The number of scores over 50 was " + str(counter))
