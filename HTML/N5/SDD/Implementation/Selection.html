Selection

A program, or algorithm, is a series of instructions. The program will go through these instructions top to bottom, step by step. Selection is when the program has reached a point where there is one or more options available to it, and it must make a decision on which set of commands to execute.

Boolean Operators

In order to make a decision, a program has to have a rule (or criteria) that tells it what decision to make. It does this by comparing one value to another, using comparison operators.

Boolean operators are used to compare two values. The result of the comparison is either true or false.

Here are a list of Boolean operators used in Python and thier meaning.

Operator Meaning
== Equal to
!= Not equal to
> Greater than
< Less than
>= Greater than or equal to
<= Less than or equal to

Note: Python uses '==' when comparing two values, rather than '='. This is because '=' is used to assign a value to a variable.

Simple Selection

Simple selection is when the program has additional code to run if a certain condition is true.

For example, a program that is calculating the total cost of a meal may display that the customer will receive a discount, but only if the cost of the meal is over £80. The program would have to look at the number stored in the mealCost variable, and if that number is over 80, it would display the additional message about the discount.

The program would look something like this:

print("Enter the cost of the meal:")

mealCost = float(input())


if mealCost > 80:

  print("You receive a 10% discount on your next meal!")


print("The cost of your meal is £" + str(mealCost))

This is a simple example, but this program would only display the message about the discount if the mealCost that was typed in was over 80. Else Statement

The program can also be told what to do if the comparison is not true. We can add an 'else:' statement onto the end for this. The code below is similar to the first set of code, but it will also tell the user if they have not spent enough to receive a discount.

print("Enter the cost of the meal:")

mealCost = float(input())


if mealCost > 80:

  print("You receive a 10% discount on your next meal!")

else:

  print("Sorry, you will not receive a discount this time.")


print("The cost of your meal is £" + str(mealCost))

Elif Statement

It is also possible to add additional conditions. In our example, the customer would receive a 10% discount on their next purchase if they spend £80, but maybe they could receive a smaller discount (5%) by spending over £50, and still nothing if they spent less than that.

To do this, we would use 'elif' (else if) to add more conditions.

print("Enter the cost of the meal:")

mealCost = float(input())


if mealCost > 80:

  print("You receive a 10% discount on your next meal!")

elif mealCost > 50:

  print("You receive a 5% discount on your next meal!")

else:

  print("Sorry, you will not receive a discount this time.")


print("The cost of your meal is £" + str(mealCost))

While it is possible to do this with multiple elif statements rather than using elif, it is more efficient to use elif. This is because the program will stop checking the conditions on the if statement once if finds a criteria that is met, rather than always having to check all of them.

Complex Selection and Logical Operators

A complex conditional statment has more than one condition. For example:

  • if age >= 16 and town == "Biggar":
  • if score < 0 or score > 0:

Remember that a conditional statement like this will only run the code inside them if the statement is True. In the first example, with age and town, both statements would have to be True for the over statement to resolve as True. In the case of the second example, either of the statements could be True for the overall statement to resolve as True.

This is done using Logical Operators - AND, OR, NOT.

AND Both conditions must be true for the statement to be true.
OR Only one condition must be true for the statement to be true.
NOT The condition must not be true for the statement to be true. For example, 'if not(found == True):' (This means that the statement would be true if the found variable were set to False)
