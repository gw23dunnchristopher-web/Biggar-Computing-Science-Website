Input Validation

What is Input Validation?

When you ask someone to enter a value into your program, you want to make sure that they enter an acceptable value. For example, if we were looking for them to enter a number, and they type in a name instead, then our program won't work. Similarly, if we ask them to enter an 8-digit password and they only type in five characters, then we can't accept that. Input validation is used to ensure that the user enters the correct data by checking what they have enter and, if it wrong, asking them to enter the data again.

We used a WHILE loop for this, because we want them to be asked to re-enter the data over and over, no matter how many times they get it wrong.

Typical pseudocode for input validation looks like this.

1. Ask the user to enter the data.
2. WHILE the data does not meet your criteria:
3. Display an error message.
4. Ask the user to enter the data again.
5. End WHILE Loop

Input Validation Examples

Here are two examples showing input validation:

The first example asks the user to enter a number between 1 and 20, and doesn't let them enter any number below 1 or above 20.

print("Enter a number between 1 and 20")

number = int(input())


while number < 1 or number > 20:

print("Error, please enter a number between 1 and 20")

number = int(input())


print("Thank you, you entered " + str(number))

The second example asks the user to enter "Y"(Yes) or "N"(No) and doesn't let them enter any other data.

print("Enter Y or N:")

answer = input()


while answer != "Y" and answer != "N":

print("Error, please enter Y or N")

answer = input()


print("Thank you, you entered " + answer)
