Assignment and Concatenation

Assignment

One of the programming constructs that you must be aware of in N5 is the assignment of a value to a variable. This is where you store a value in a variable so that it can be used later in the program.

The assignment operator is the = symbol.

Here are some examples of assignment:

age = 10

The above code assigns the integer 10 to a variable called 'age'.

name = "Bob"

This code assigns the string 'Bob' to the variable name. Note that when using a string, you must surround the string with speech marks. This is to differentiate it from other data types, since strings can use numbers, letters or symbols.

height = 1.8

This code assigns the float (real number) 1.8 to the variable height.

found = False

The code shown here assigns the Boolean value 'False' to a variable called 'found'. As this is a Boolean value, it means that the variable can be set either to True or False, indicating whether whatever you are looking for has been found or not.

The input() function

This function pauses the program and waits for the user to type something in. When using the input function, you must remember two things:

  • You must assign the input() function to a variable. If you don't then whatever you enter will be lost.
  • The input() function automatically converts whatever you enter into a string. If you want to enter a number, then you must use typecasting (int() or float()) together with the input() function.

Here are a couple of examples of using the input() function:

name = input()

This code assigns whatever the user enters to the variable 'name'. The name entered will be stored as a string.

age = int(input())

Here we have entered a number. The input function has been placed inside the int() type casting function. The input function will take in whatever the age entered as a string ("10")# and then the int() function will convert it to an integer (10).

height = float(input())

A similar thing has occured here, only this time the string taken in by the input() function will be converted into a real number with a decimal point.

One more point to note about the input() function is that any string placed in between the brackets will be displayed as a question for the user when the program is waiting for them to enter something:

name = input("Please enter your name: ")

This code will display the string "Please enter your name: " and then wait for the user to enter thier name. The name entered will be stored in the variable 'name'.

You can use this or just use the print() function to display the question.

The print() function

The 'print()' function is used to display a message on the screen. The message to be display goes in between the brackets. It should be noted that it cannot display multiple datatypes unless they are separated by commas.

Here are a few examples:

print("Hello World")

This code will display the message 'Hello World' on the screen. As the message is a string, it is surrounded by speech marks.

print(10)

This code will display the integer 10 on the screen. As it is an integer, it does not need speech marks.

print(True)

This will print the word True on the screen, it does need speech marks as it is a Boolean value.

Concatenation

Concatenation is the process of joining two strings together. This is done by using either the + or the , symbol.

The + symbol only works with strings, so any integers or floats used in the message must be converted to strings using the str() function. Using + does not automatically create a space between the two strings.

Using a comma instead of + will automatically create a space between the two strings. Additionally, you can also concatenate different datatypes without having to convert them to strings.

We typically use concatenation to insert values stored in variables into a message. Look at the code below:

name="Bob"

print("Your name " + name + ".")

This code will display the message 'Your name Bob.' The name "Bob" has been stored in the variable 'name', and so using 'name' here will insert the value stored in 'name' into the message.

name = "Bob"

print("Your name is", name, ".")"

This code uses the commas instead. It will display the message "Your name is Bob .". There is an additional space before the full stop because the comma automatically creates one.

age = 10

print("You are " + str(age) + " years old")

This code will display the message 'You are 10 years old' on the screen. The number 10 has been stored in the age variable, so it can be used in the message. The str() function has been used to convert the integer 10 into a string.

age = 10

print("You are", age, "years old")

This is the same code using commas, note that there is no need to put the spaces in yourself, or to convert age to a string using str().

The input() function can be used to get the user to enter information to be used later, for example in a message.

print("Enter your age:")

age = int(input())

print("You are " + str(age) + " years old.")

This code will:

  • display the message 'Enter your age:' on the screen.
  • get the user to enter thier age.
  • convert the string entered into an integer.
  • display the message "You are (age) years old." on the screen. With (age) replaced by whatever number the user entered.
