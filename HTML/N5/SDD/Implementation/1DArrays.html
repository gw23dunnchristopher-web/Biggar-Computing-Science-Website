1D Arrays

An array is a data structure used to store a number of values of the same data type.

  • Arrays can hold multiple values, but all of the values held in the array must be the same data type (all strings, all integers, etc).
  • Python does not have arrays natively, instead it uses lists to simulate an array.
  • Each array has a number of elements, this is the number of values that can be stored in the array.
  • The first value stored in the array is element 0, the second is element 1, the third is element 2, and so on.

Initialising an Array

We initialise an array using square brackets as shown below:

Code Explanation
num = [0] * 4 This will create an array of integers with 4 elements.
names = [“”] * 15 This will create an array of strings with 15 elements.
prices = [0.0] * 5 This will create an array of real numbers with 5 elements.
answers = [False] * 10 This will create an array of Boolean values with 10 elements.

Inserting values into an Array

To assign a value to a specific element in an array, we use the name of the array and then the element number we want to insert the value into in square brackets.

For example:

names[1] = "Bob"

names 0 1 2 3
" " "Bob" " " " "

The code next to the table above would insert the name "Bob" into element 1 of an array called 'names'.

Using a fixed loop to insert values

We can use a fixed loop to insert values into an array. See the Loops section for more information on fixed loops.

The code below will ask the user to enter 6 names. It will then insert the names into an array.

names = [""] * 6


for x in range(6)

  print("Enter name of student", (x + 1))

  scores[x] = int(input())

This code will:

  • initialise an array of strings called 'names' with 6 elements.
  • loop 6 times (with x increasing by 1 with each loop - 0, 1, 2, 3, 4, 5)
  • ask the user to enter a name, and place it in the element with the same number as the current iteration number (x) before going back to the start of the loop.

Using a conditional loop to insert values

The code below will use a conditional loop to get the user to enter 6 scores. If any of the scores entered are less than 10, it will stop the loop and no more scores will be entered.

scores = [0] * 6

counter = 0


while counter < 6 and score[counter-1] < 10:

  print("Enter score", (counter+1)

  scores[counter] = int(input())

  counter = counter + 1

This code will:

  • initialise an array of integers called 'scores' with 6 elements.
  • use a counter to check if all of the numbers have been entered.
  • exit the loop of the number entered is less than 10.
  • increase the counter using a running total.

Doing it this way allows your loop to have other exit conditions, so that you can stop entering values into the elements of the array before it is completely filled.

Displaying the values in an array

To display the value stored in one element of the array, you just have to use the print function, the name of the array and square brackets containing the element.

For example, look at the following array:

print(names[2])

names 0 1 2 3
"Tom" "Bob" "Peter" "Josh"

The code above would display the name "Peter" because it is stored in element 2 of the array.

Displaying all the values in the array (with a loop)

The code below will display all of the values in the array using the incremental number (x) in place of the element number.

for x in range(4):
  print(names[x])
