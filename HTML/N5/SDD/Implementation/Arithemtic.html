Arithmetic Operations

At N5 level, our programs must also be able to perform arithmetic operations - addition, subtraction, multiplication, division and exponentiation (to the power of).

The following table shows the symbols used in Python for the different arithmetic operations, and thier meaning.

Arithmetic Operator Meaning
+ Addition - Adds numbers together.
- Subtraction - Subtracts one number from another.
* Multiplication - Multiplies two numbers together.
/ Division - Divides one number by another.
** Exponentiation - One number to the power of another.

Here are some examples:

print(5+5)

The code above would display the number 10.

print(10-8)

This code would display the number 2.

print(5*5)

This code would display the number 25.

print(10/2)

This code would display the number 5.

print(5**2)

This code would display the number 25 (5 to the power of 2).

Variables can also be used with arithmetic operators. For example, the code below will ask the user to enter two numbers and then add them together.

num1 = int(input("Enter a number: "))

num2 = int(input("Enter another number: "))

total = num1 + num2


print("The total is", total)

Usually, we would only really use these operators with integers and floats, but some of them can be used with strings as well. For example, using the + operator can add two strings together (concatenation), and using the * multiply will display the same string multiple times. You cannot add a numeric value to a string.

BODMAS

BODMAS stands for Brackets, Orders, Division, Multiplication, Addition.

This is the order that computers perform calculations in when there are multiple calculations to be done in the same line.

For example:

totalCost = 30 * adultTickets + 15 * childTickets

This code will calculate the total cost of some tickets, assuming that the adult tickets cost £30 and the children's tickets cost £15. The multiplications will performed here first, before the addition, multiplying the number of adults by 30 and the number of children by 15 before adding those two multiplied values together.
