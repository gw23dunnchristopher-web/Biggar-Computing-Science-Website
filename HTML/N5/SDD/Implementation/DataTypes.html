Variables and Data Types

What is a variable?

A variable is like a container or box that can hold a value, such as a number, a word, or any kind of data. In programming or maths, we use variables to store information that might change or be used later.

For example, if you're working out how much pocket money you get every week, you might use a variable to store the amount. Let's say you get £10 every week – you could call this variable "pocketMoney". If next week you get £12, you can change the value of the variable.

In maths, variables are usually represented by letters like x or y. So, if you have the equation "x + 3 = 7", the variable "x" is the unknown value that you're trying to find. In this case, x would be 4 because 4 + 3 equals 7.

So, a variable helps us keep track of parts of our program that can change or need to be calculated

Variable Names

When you're naming a variable, there are a few important rules to keep in mind. These rules make sure that your program works properly and is easy to understand.

  1. Start with a letter - A variable name must begin with a letter, not a number. So, you can name a variable "score" or "totalAmount", but you can't start it with "2ndPrize" or "3points".
  2. Only letters, numbers, and underscores - After the first letter, you can use letters, numbers, and underscores ( _ ). So, "total_earnings", "x1", and "studentScore2" are all fine. But, you can't use spaces or special characters like @, $, or % in a variable name.
  3. Keep it meaningful - While not a strict rule, it's a good idea to name your variables something that makes sense. For example, use "age" for someone's age or "price" for the price of something. This helps make your code easier to read and understand.
  4. Case-sensitive - Variable names are case-sensitive, which means "score" and "Score" would be considered two different variables.
  5. No spaces - You can't put spaces in variable names. If you need to separate words, use an underscore ("_") or use camel case, where the first letter of each word after the first one is capitalised (e.g. "totalAmount" or "total_amount").

Data Types

There are five data types that you need to be aware of at the N5 level:

  • Character - a variable that only stores a single character.
  • String - a varaiable that stores a number of characters (letters, symbols and numbers).
  • Numeric (Integer) - a variable that stores a whole number (without the decimal part).
  • Numeric (Real) - a variable that stores a number with the decimal part. In Python, Real numbers are called ""floats""
  • Boolean - a variable that can store two values, either True or False.

Type Casting

In Python, you need to know functions that will change one type of data into another. For example, you cannot compare a string to a number to check if the string is higher or lower, even if that string contains a number (you could not type 5 > "2", for example). The string must be changed into a number using type casting.

Converting Into Function Usage Example
Character chr() asciiChar = chr(77) (This would make the value in the asciiChar variable "M", since 77 is M in ASCII.
String str() total = str(27) (This would make the value in total "27". Note the speech marks, meaning it is now a string.)
Numeric (Integer) int() score = int("27") (This would change the value in the score variable to 27)
Numeric (Real) float() price = float(“1.99”) (This would change the value in the price variable to 1.99)
Boolean bool answer = bool(“True”) (This would set the value in the answer variable to True, this function is rarely used)
