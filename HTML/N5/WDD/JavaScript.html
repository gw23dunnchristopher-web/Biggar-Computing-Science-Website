What is JavaScript?

JavaScript is a programming language that is used to make webpages interactive. It is what's known as a 'client-side' language, which means that it is run on the user's computer, rather than on the server. It is used to add functionality to webpages, such as changing the content of a page, changing the style of a page, and responding to user input.

Much like CSS, JavaScript can either be placed in the <head> (in between <script> tags) or in a separate .js file which is then linked to the webpage. The JavaScript itself consists of functions which are then linked to sections of the webpage in the html code, and called using events.

What you need to know

For the N5 level, you do not need to know a lot about JavaScript. You do not need to create JavaScript functions, or add them to a webpage. You just need to know how and where the functions are called and where the events are placed. There are two events you need to know about at N5:

  • onmouseover - This will call a JavaScript function when the user's cursor moves onto the element.
  • onmouseout - This will call a JavaScript function when the user's cursor moves away from an element.

Both of these are used as attributes inside the tag for the element that you want to call the function. Here is an example where a function called 'brighten' is being called when the user moves their cursor onto the div, and calls a 'darken' function when they move it away again.

<div onmouseover="brighten()" onmouseout="darken()">

</div>
