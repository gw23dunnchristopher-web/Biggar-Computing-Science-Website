File Formats and Compression

File Formats

At the N5 level you need to be aware of the difference between different file formats for images, audio and video files that can be used as content for a website.

Image File Formats

There are several different file formats that can be used for images on a website. Below is a table that compares the different formats. The formats shown here are all bitmap based, so the all have a number of bits used for each pixel to indicate the number of colours that can be represented.

Format Details Example
JPG (Joint Photographic Expert Group)
  • uses lossy compression - level of compression can be varied.
  • no animation.
  • no transparency.
  • colour depth - usually 24bits used per pixel, so 16,777,216 colours are available (224).
PNG (Portable Network Graphic)
  • uses lossless compression.
  • no animation.
  • has different levels of transparency, so some parts of the image can be semi-transparent.
  • colour depth - 24 or 48 bits.
GIF (Graphics Interchange Format)
  • uses lossless compression.
  • can be animated.
  • A single level of transparency, so parts of the image can be transparent.
  • colour depth - 8, so 256 colours available (28)
  • Used for simple graphics.

Audio File Formats

There are several file formats used for Audio - WAV, MP3, FLAC, AIFF, OGG, etc. For N5, we will just look at two of these, WAV and MP3. You can listen to examples of a WAV file and an MP3 file below - they sound almost exactly the same, but the MP3 file is a tenth of the size of the WAV file! Watch how much faster the MP3 loads.

Format Details Example
WAV (WAVeform Audio File Format)
  • usually uncompressed.
  • high quality.
  • can be saved with different sampling rates (e.g. 44.1 KHz).
  • can be saved using different bit rates (e.g. 16 bit).
MP3 (MPEG (Moving Picture Experts Group) Audio Layer 3)
  • compressed audio file.
  • high quality - compression removes sounds that can’t be heard by humans.
  • small file size - 10% of the size of a wav file.

Video File Formats

Much like with audio file formats, there are a large number of different formats used for video files. We will just focus on two of them - AVI and MP4.

Format Details Example
AVI (Audio Video Interleave)
  • larger file size (less compression).
  • high video quality.
  • not ideal for streaming over the internet.
  • more control over the bitrate of doing video editing.
MP4 (MPEG (Moving Picture Experts Group)Part 14)
  • compressed video file.
  • still good quality, even with compression
  • small file size.
  • Works on almost all devices and platforms.
  • Great for online streaming.

Factors Effecting File Size

Images

  • Resolution - The more pixels are used in the image, the higher the file size of the image will be.
  • Colour Depth - Images with a larger colour depth use a greater number of bits to represent the colour of each pixel. The quality will be better since more colours are available, but the file size will be larger.

Audio

  • Sample Rate - Higher sample rates (e.g., 44.1 kHz vs 22 kHz) capture more audio details, increasing the file size.
  • Bit Depth - Greater bit depth (e.g., 16-bit vs 24-bit) offers higher sound quality but results in larger files.
  • Channels - Stereo (2 channels) audio files are smaller than surround sound (e.g., 5.1 channels) audio files.

Videos

  • Resolution - Higher resolutions (e.g., 1080p vs 4K) lead to bigger file sizes.
  • Framerate - More frames per second (e.g., 60fps vs 30fps) increase file size.
  • Bit Rate - Higher bit rates ensure better quality but result in larger file sizes.
  • Audio Integration - Embedded audio increases file size depending on its quality and compression.
  • Duration - Longer videos naturally have larger sizes.

Compression

What is Compression?

Compression is a technique used to reduce the size of data or files without completely losing their functionality or purpose. This process is widely applied across images, audio, video, and even text to make them easier to store, transfer, or process.

It works by identifying patterns or redundancies within the data and replacing them with more efficient representations. There are two types of compression:

  • Lossless Compression - This method tries to reduce the size of the file without losing any data or quality. It is commonly used for text files, images, and audio files where preserving the original data is crucial.
  • Lossy Compression - This method reduces file size by discarding some data, which results in a loss of quality. It is often used for images and video files where a slight loss in quality is acceptable to achieve a significant reduction in file size.

Why use compression?

  • Many files are large – e.g. graphic files, audio files or video files
  • Compressed files use less space on backing storage and in RAM
  • Compressed files are quicker to transfer across a network
