Evaluating a Website

After the website has been tested to make sure everything works, it must be evaluated to make sure that it is fit for purpose. A website is considered to be fit for purpose if it meets the:

  • End-User Requirements determined in the Analysis phase.
  • Functional Requirements determined in the Analysis phase.

These requirements can be used as a checklist to determine whether or not the site meets the criteria. If the website does not meet all of these requirements then it is considered to be unfit for purpose it may be necessary to go back to the implementation or even the design stage to rework the site.
