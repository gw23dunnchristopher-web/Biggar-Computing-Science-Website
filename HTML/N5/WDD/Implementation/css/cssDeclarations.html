CSS Declarations

CSS Rules can change the style of your website in a variety of ways. For the N5 level, you have to be aware of how to change the size, colour, alignment and style of the font, and the background colour. You can find examples of these declarations in the table below:

Declaration Description Example
font-size
  • Changes the size of the font.
  • Can be set using pixels (px) or ems (em). You can also use a percentage to change the size of the font based on its original size.
  • font-size:12px; - Sets the font size to 12 pixels.
  • font-size:1.5em; - Sets the font size to 1.5 times the original size.
  • font-size:150%; - Sets the font size to 150% of the original size.
color
  • Changes the colour of the font.
  • Can be set using a colour name, a hex code, or an RGB value.
  • color:red; - Sets the font colour to red.
  • color:#ff0000; - Sets the font colour to red using a hex code.
  • color:rgb(255,0,0); - Sets the font colour to red using an RGB value.
font-family
  • Changes the font family.
  • Can be set using a font name, or a list of font names.
  • font-family:Arial; - Sets the font family to Arial.
  • font-family:Arial, Helvetica, sans-serif; - Sets the font family to Arial, Helvetica, or a sans-serif font.
  • font-family:"Times New Roman", Times, serif; - Sets the font family to Times New Roman, Times, or a serif font.

Note that if there are spaces in the font name, you have to use speech marks.
text-align
  • Changes the alignment of the text.
  • Can be set to left, right, center, or justify.
  • text-align:left; - Sets the text alignment to left.
  • text-align:right; - Sets the text alignment to right.
  • text-align:center; - Sets the text alignment to center.
  • text-align:justify; - Sets the text alignment to justify.
background-color
  • Changes the background colour of the element.
  • Can be set using a colour name, a hex code, or an RGB value.
  • background-color:red; - Sets the background colour to red.
  • background-color:#ff0000; - Sets the background colour to red using a hex code.
  • background-color:rgb(255,0,0); - Sets the background colour to red using an RGB value.

Note that if you use this on a container element like body, div, or html, it will change the background colour of the container. If used on a text tag like <p> or <h1> it will change the colour of the line that the text is on.

Example CSS Rule

The <div> tag below has been style using the following CSS rule:

div {

    background-color:lightblue;

    color:darkgreen;

    font-size:20px;

    font-family:"Times New Roman", Times, serif;

    text-align:center;

}

Note that because text styles have been used on the <div> tag, these styles have been used on the text inside the <div>.

State the colour of the ‘Wellbeing’ heading when displayed in a browser.

 1

White - while the wellbeing heading does not have a rule that changes its colour by itself, it is inside a <div> that has the 'wellbeing' class. In the CSS, anything with this class has its text coloured white (it's the '.wellbeing' rule).
