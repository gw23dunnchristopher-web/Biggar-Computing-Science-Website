CSS Basics

What is CSS?

CSS, or Cascading Style Sheets, is a language used to style and format the appearance of webpages. While HTML is responsible for the structure and content of the page, CSS defines how it looks. With CSS, you can control the colours used on your website, the style of the font, text alignment, background colours and many other elements. The 'Cascading' part of the name refers to hwo styles are applied in a specific order of importance. There are three kinds of CSS, which I will list here in order:

  • External CSS - CSS rules are stored in the an external .css file which is then linked to the webpage. This is a really useful kind of CSS as it can allow you to apply the same styles to multiple webpages on your website. It is the least important kind of CSS to the browser, as its styles will be overwritten by internal and inline CSS.
  • Internal CSS - This kind of CSS is used to apply style to a single webpage. The style rules are kept in the <head> tag, inside the <style> tag. It will override external CSS, but not inline CSS.
  • Inline CSS - It is inline because the styles are inside one of the tags on the webpage - the styles only affect that tag and nothing else. This will override External and Internal CSS.

Internal and External CSS rules

External CSS

The file containing External CSS must be linked to your webpages. The link to the CSS file goes in the <head> tag of the webpage. The link tag has two attributes:

  • rel - This attribute is set to "stylesheet".
  • href - This attribute is the file path to the CSS file.
  • type - This attribute is set to "text/css".

Here is an example of a link to an external CSS file:

<link rel="stylesheet" href="/CSS/styles.css" type="text/css">


The rules contained in the CSS file consist of a selector and declarations. The selector denotes which tag will be affected by the rule, and the declarations are the elements that the rule will change, such as the colour of the text or its alignment. Internal CSS

Internal CSS is written inside the <style> tags, which are inside the <head> tags. Here is an example of internal CSS:

<style>

    p {

        color:red;

        text-align:center;

        font-size:14px;

    }

</style>

This will effect all of the <p> tags on the page that has the style tags and the rule itself in the head.

Inline CSS

Inline CSS is written inside the tag that you want to style. The style attribute is used to define the CSS rules. Here is an example of inline CSS:

<p style="color:red; text-align:center; font-size:14px;">Styled text</p>

This will only effect the <p> tag that it is inside.

(iii)

Explain why the line of code below is included in every page of the website.

<link rel = "stylesheet" type="text/css" href="ChillStyle.css">

 1

This line of code links the external CSS file to the webpage. The CSS file contains the style rules that will be applied to the webpage. This line of code is included in every page of the website so that the same styles are applied to every page.
