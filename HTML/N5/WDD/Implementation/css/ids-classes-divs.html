IDs, Classes and Divs

What is a div?

A div is a container element that is used to group other elements together. It is a block level element, which means that it will take up the full width of its container. It is a very useful element as it allows you to apply styles to a group of elements at once. It is also useful for layout purposes, as it allows you to create sections on your webpage.

Here is an example of the code used to create two divs:

<div>

    <p>This is a paragraph inside a div.</p>

</div>

<div>

    <p>This is another paragraph inside a div.</p>

</div>

IDs and Classes

IDs and Classes are used to apply styles to specific elements. They are both attributes of the tag, and are used in the CSS to select the element. The difference between them is that an ID can only be used once on a page, while a class can be used multiple times. IDs are used to apply styles to a single element, while classes are used to apply styles to a group of elements.

Here is an example of the code used to create two divs with IDs and Classes:

<div id="div1">

    <p>This is a paragraph inside a div with an ID.</p>

</div>

<div class="div2">

    <p>This is another paragraph inside a div with a class.</p>

</div>

IDs and CSS

CSS can be used to style a div with an ID, to do this the CSS rule would include the ID with a hash tag (#) in front of it as shown below:

#div1 {

    background-color:lightgreen;

    color:darkblue;

    font-size:15px;

    font-family:Calibri;

    text-align:right;

}

If the ID (div1) is applied to a div as shown above, you would get the following output:

This is a paragraph inside a div with an ID.

IDs denote a single location on the page, so they can only be used in one place. You cannot use the same ID for multiple divs, for example.

Classes and CSS

CSS can be used to style a div with a class, to do this the CSS rule would include the class with a dot (.) in front of it as shown below:

.div2 {

    background-color:lightblue;

    color:darkred;

    font-size:20px;

    font-family:Arial;

    text-align:center;

}

If the class (div2) is applied to a div as shown above, you would get the following output:

This is another paragraph inside a div with a class.

Classes can be used multiple times on a page, so you can use the same class for multiple divs, for example.
