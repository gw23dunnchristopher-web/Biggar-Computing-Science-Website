What are anchor tags?

Anchor tags, the <a> tag, are used to create hyperlinks on your website. There are several types of link that can be created using anchor tags:

  • External Hyperlinks - These are links to a page on another website.
  • Internal Hyperlinks - These are links to other pages on the same website.
  • Jump (Anchor) Hyperlinks - These are links to a specific section of a webpage.

Hyperlinks like this use two kinds of addressing, absolute and relative.

  • Absolute Addressing - This is the full URL of the page you want to link to. This is used for external hyperlinks.
  • Relative Addressing - This is the path to the page you want to link to, relative to the current page. This is used for internal hyperlinks.

Jump links use an ID to link to a section of a webpage that is given that ID.
