Lists

Types of List

There are two types of list that are used in HTML - ordered lists and unordered lists.

  • Ordered Lists - These are numbered lists, like a list of instructions numbered 1, 2, 3, 4 or a, b, c, d.
  • Unordered Lists - These lists have no particular order, they are just bullet points.

List in HTML

Lists in HTML are created using the <ul> and <ol> tags. The <ul> tag is used for unordered lists, and the <ol> tag is used for ordered lists. Each item in the list is created using the <li> tag.

Here is a table showing the code for lists and how they would appear in HTML:

</ol>
