Image Tags

Displaying Images on a Webpage

Webpages without pictures can be very boring. To add pictures to your website, you can use the <img> tag.

The image tag has four attributes (an attribute allows you to modify a tag to change what is shown on the webpage):

  • <src> - This attribute tells the webpage where the image is located. If it is saved to the same folder as the webpage this will be the image name and file type. However, if it is saved in a different folder, then this attribute will have to include the file path of the image.
  • <alt> - If there is a problem and the image cannot be displayed, this attribute displays some text instead.
  • <height> - Controls the height of the image. This can be a specific number of pixels ("100px"), or a percentage of its original size ("50%").
  • <width> - If there is a problem and the image cannot be displayed, this attribute displays some text instead.

As it is displayed on the webpage, <img> tags are used inside the <body> tag in the HTML code. Here is an example if an image tag with its associated attributes, the code has been used in this website to display the image to the right. The image tag does not have a closing tag.

<img src="/Images/N5/WDD/cat.jpg" alt="Image of a Cat" height="300px" width="300px"/>

Note that because this image is inside a different folder, the file path is included in the source - "/Images/N5/WDD/". This indicates that from the root directory of the website, the image is contained in the WDD folder, which is inside the N5 folder, which is inside the Images folder.

The next example uses an image that is on a different website. As you can see it uses the website address:

<img src="https://thumbs.dreamstime.com/b/cute-cat-portrait-square-photo-beautiful-white-closeup-105311158.jpg" alt="Image of a Cat" height="300px" width="300px"/>
