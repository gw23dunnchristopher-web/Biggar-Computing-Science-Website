Audio and Video Tags

Audio and Video

Websites can also use tags to play audio or display video. Video is probably the more common of the two, as it is more engaging than audio. However, audio can be used to play music or sound effects.

One way to display video on a website is to embed it using iframes. Most websites like YouTube have a way to share videos by copying the embed code and pasting it directly into your HTML code. However, in this section we are going to look at using the <audio> and <video> tags to display those elements.

The Audio Tag

The <audio> tag has an opening and closing tag, with a <source> tag in between the two. The <source> tag has two attributes: src and type. The src attribute is the file path to the audio file, and the type attribute is the file type of the audio file. The <audio> itself has several attributes that you can use to control how the audio is played:

  • autoplay - If this attribute is present, the audio will start playing as soon as the page loads.
  • controls - If this attribute is present, the audio will have controls to play, pause, and adjust the volume.
  • loop - If this attribute is present, the audio will loop indefinitely.

The <video> tag is very similar to the <audio> tag. It has an opening and closing tag, with a <source> tag in between the two. The main difference is that with a video tag, you can control the width and height of the video, using those attributes.
