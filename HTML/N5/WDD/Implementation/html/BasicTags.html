Basic HTML Tags

What are HTML Tags?

HTML, or Hypertext Markup Language, is the coding language that websites are written in. An HTML document is essentially just a text file, but it is written in a specific way that can be interpreted as a webpage by internet browsers like Chrome, Edge, Firefox and Safari. HTML files use 'tags' to order to tell the browser program what to display on the website and in what order. Most tags have an opening tag and a closing tag, this tells the browser where the start and end of that element on the webpage is.

Here is a list of basic tags that you might see in an HTML document.

Opening Tag Closing Tag Use
<!DOCTYPE html> none This tag appears at the top of an HTML document. It is used to declare the document type and the version of HTML being used. Developers use it to make sure their website renders correctly. It does not have a closing tag.
<html> </html> Signifies the beginning and end of an HTML document. All of the other tags go (apart from the !DOCTYPE tag), go inside the html tags.
<head> </head> This tag identifies the webpage's header - the part of the webpage that does not appear in your browser. It is where you place all the metadata for your website (the data about the site). The title goes here, since that appears in the tab for the webpage and not on the site itself. This is also where you can place links to stylesheets, and internal CSS and JavaScript functions.
<title> </title> The title tags go inside the header tag, and define the name of the webpage. This name will appear in the tab at the top of the browser when the webpage is open.
<body> </body> Defines the main body of your webpage. Everything in between the two body tags will be displayed on the webpage.
<h1> </h1> This is largest heading tag. It goes inside the body tag so that the text will appear correctly on the webpage. By default the text shown by this tag is a bold Sans-serif font. Its default size is 32px.
<h6> </h6> This is smallest heading tag. By default the text shown by this tag is a bold Sans-serif font. Its default size is 10-12px depending on the browser.
<p> </p> Any text in between these two tags will appear as standard paragraph text on your web page. This will appear as default san-serif non-bold font with a size of about 16px.
